# <a name="inventory-aging-report-discrepancies"></a>Készletkorosítási jelentés eltérései

## <a name="symptoms"></a>Tünetek

A **Készletkorosítási jelentés** különböző értékeket jelenít meg különböző tárolási dimenziókban (például hely vagy raktár).

## <a name="resolution"></a>Megoldás

A jelentési logikával kapcsolatos további tudnivalókat lásd: [Készletkorosítási jelentés példák és logika](/dynamics365/supply-chain/cost-management/inventory-aging-report.md).
