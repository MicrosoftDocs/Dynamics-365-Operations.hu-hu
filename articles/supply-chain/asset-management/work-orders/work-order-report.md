[!include [banner](../../includes/banner.md)]


<span data-ttu-id="3d744-104">Létrehozhat egy munkarendelési jelentést, amely részletes adatokat jelenít meg egy vagy több kijelölt munkarendeléssel kapcsolatban.</span><span class="sxs-lookup"><span data-stu-id="3d744-104">You can generate a work order report that shows detailed information about one or more work orders that you select.</span></span>

1. <span data-ttu-id="3d744-105">Válassza az **Eszközkezelés** > **Jelentések** > **Munkarendelések** > **Munkarendelési jelentés** elemet.</span><span class="sxs-lookup"><span data-stu-id="3d744-105">Select **Asset management** > **Reports** > **Work orders** > **Work order report**.</span></span>

2. <span data-ttu-id="3d744-106">A jelentésbe foglalandó részletek meghatározásához válassza ki az „igen”/„nem” beállítást a választógombokon.</span><span class="sxs-lookup"><span data-stu-id="3d744-106">Select "Yes"/"No" on the toggle buttons as you require, to determine the details to be included in the report.</span></span>  
    - <span data-ttu-id="3d744-107">A **Nyomtatási beállítások** részben kiválaszthatja, hogy szerepeljenek-e a kapcsolódó karbantartási feladattípus-beállítás mellékletei.</span><span class="sxs-lookup"><span data-stu-id="3d744-107">In the **Print settings** section, you can select whether to include attachments from the setup of the related maintenance job type.</span></span>

3. <span data-ttu-id="3d744-108">A **szerepeltetni kívánt rekordok** gyorslapon a jelentés tartalmát **Munkarendelés** szerint szűrheti.</span><span class="sxs-lookup"><span data-stu-id="3d744-108">On the **Records to include** FastTab, you can filter the contents of the report by **Work order**.</span></span>

4. <span data-ttu-id="3d744-109">A **Futtatás a háttérben** gyorslapon szükség szerint kötegelt feladatként is beállíthatja a munkarendelési jelentéskészítést.</span><span class="sxs-lookup"><span data-stu-id="3d744-109">On the **Run in the background** FastTab, you can set up generation of the work order report as a batch job, as you require.</span></span>
