12. <span data-ttu-id="7da14-117">Kattintson a Mentés gombra.</span><span class="sxs-lookup"><span data-stu-id="7da14-117">Click Save.</span></span>
13. <span data-ttu-id="7da14-118">Zárja be a lapot.</span><span class="sxs-lookup"><span data-stu-id="7da14-118">Close the page.</span></span>
14. <span data-ttu-id="7da14-119">Kattintson az Eladási árak lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="7da14-119">Click Sales prices.</span></span>
15. <span data-ttu-id="7da14-120">Kattintson az Eladási ár – óra lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="7da14-120">Click Sales price - hour.</span></span>
16. <span data-ttu-id="7da14-121">Kattintson az Új lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="7da14-121">Click New.</span></span>
