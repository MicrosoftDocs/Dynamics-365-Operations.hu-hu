14. <span data-ttu-id="7f851-119">Kattintson az Eladási árak lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="7f851-119">Click Sales prices.</span></span>
15. <span data-ttu-id="7f851-120">Kattintson az Eladási ár – óra lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="7f851-120">Click Sales price - hour.</span></span>
16. <span data-ttu-id="7f851-121">Kattintson az Új lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="7f851-121">Click New.</span></span>
