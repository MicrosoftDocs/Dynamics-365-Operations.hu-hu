13. <span data-ttu-id="5d586-118">Zárja be a lapot.</span><span class="sxs-lookup"><span data-stu-id="5d586-118">Close the page.</span></span>
14. <span data-ttu-id="5d586-119">Kattintson az Eladási árak lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="5d586-119">Click Sales prices.</span></span>

16. <span data-ttu-id="5d586-121">Kattintson az Új lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="5d586-121">Click New.</span></span>
17. <span data-ttu-id="5d586-122">Adja meg a dátumot az Érvényesség dátuma mezőben.</span><span class="sxs-lookup"><span data-stu-id="5d586-122">In the Effective date field, enter a date.</span></span>
