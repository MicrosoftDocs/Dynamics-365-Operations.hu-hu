# <a name="set-up-subscription-groups"></a>Előfizetési csoportok beállítása 

[!include [banner](../includes/banner.md)]


1.  Kattintson a következőkre: **Szolgáltatáskezelés**\>**Beállítás**\>**Szolgáltatási előfizetések**\>**Előfizetési csoportok**.

2.  Nyomja le a CTRL+N billentyűkombinációt, vagy kattintson az **Új** gombra, ha új előfizetési csoportot szeretne létrehozni.

3.  Az **Előfizetési csoport** mezőbe írja be az új előfizetés-csoport kívánt, jól felismerhető nevét.

4.  A **Számlázás** mezőben válasszon ki egy számlázási gyakoriságot az előfizetéshez. A számlagyakoriságot egy időszakkód jelöli, amelynek beállítása az **Időszaktípusok** képernyőn történik.

5.  A bevétel elhatárolásához előfizetési tranzakciókra vonatkozóan, jelölje be a **Bevétel elhatárolása** jelölőnégyzetet, majd az **Elhatárolás** mezőben válassza ki a gyakoriságát. Ahogy a számlagyakoriságnak, az elszámolási gyakoriságnak is egy időszakkód az alapja, amelyet az **Időszakok** képernyőn lehet beállítani.
