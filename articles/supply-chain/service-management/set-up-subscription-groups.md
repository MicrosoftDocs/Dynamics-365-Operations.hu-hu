# <a name="set-up-subscription-groups"></a><span data-ttu-id="be6f5-103">Előfizetési csoportok beállítása</span><span class="sxs-lookup"><span data-stu-id="be6f5-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="be6f5-104">Lépjen ide: **Szolgáltatáskezelés** \> **Beállítás** \> **Szolgáltatási előfizetések** \> **Előfizetési csoportok**.</span><span class="sxs-lookup"><span data-stu-id="be6f5-104">Go to **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="be6f5-105">Válassza ki az **Új** lehetőséget egy új előfizetési csoport létrehozásához.</span><span class="sxs-lookup"><span data-stu-id="be6f5-105">Select **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="be6f5-106">Az **Előfizetési csoport** mezőbe írja be az új előfizetés-csoport kívánt, jól felismerhető nevét.</span><span class="sxs-lookup"><span data-stu-id="be6f5-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="be6f5-107">A **Számlázás** mezőben válasszon ki egy számlázási gyakoriságot az előfizetéshez.</span><span class="sxs-lookup"><span data-stu-id="be6f5-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="be6f5-108">A számlagyakoriságot egy időszakkód jelöli, amelynek beállítása az **Időszaktípusok** képernyőn történik.</span><span class="sxs-lookup"><span data-stu-id="be6f5-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>

5.  <span data-ttu-id="be6f5-109">A bevétel elhatárolásához előfizetési tranzakciókra vonatkozóan, jelölje be a **Bevétel elhatárolása** jelölőnégyzetet, majd az **Elhatárolás** mezőben válassza ki a gyakoriságát.</span><span class="sxs-lookup"><span data-stu-id="be6f5-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="be6f5-110">Ahogy a számlagyakoriságnak, az elszámolási gyakoriságnak is egy időszakkód az alapja, amelyet az **Időszakok** képernyőn lehet beállítani.</span><span class="sxs-lookup"><span data-stu-id="be6f5-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
