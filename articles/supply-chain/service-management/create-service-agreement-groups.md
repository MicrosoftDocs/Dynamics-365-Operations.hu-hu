# <a name="create-service-agreement-groups"></a><span data-ttu-id="07557-103">Szolgáltatásiszerződés-csoportok létrehozása</span><span class="sxs-lookup"><span data-stu-id="07557-103">Create service agreement groups</span></span> 

[!include [banner](../includes/banner.md)]

<span data-ttu-id="07557-104">A szolgáltatásiszerződés-csoportok jól hasznosíthatók a szolgáltatási szerződések rendezésére és szűrésére.</span><span class="sxs-lookup"><span data-stu-id="07557-104">Service agreement groups are useful for sorting and filtering service agreements.</span></span>

1. <span data-ttu-id="07557-105">Kattintson a **Szolgáltatáskezelés** \> **Beállítás** \> **Szolgáltatási szerződések** \> **Szolgáltatásiszerződés-csoportok** lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="07557-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service agreement groups**.</span></span>
2. <span data-ttu-id="07557-106">Hozzon létre egy új szolgáltatásiszerződés-csoportot az **Alt+N** billentyűkombináció lenyomásával.</span><span class="sxs-lookup"><span data-stu-id="07557-106">Press **Alt+N** to create a new service agreement group.</span></span>
3. <span data-ttu-id="07557-107">A **Szolgáltatásiszerződés-csoport** mezőben adja meg szolgáltatásiszerződés-csoport azonosítóját.</span><span class="sxs-lookup"><span data-stu-id="07557-107">In the **Service agreement group** field, enter an ID for the service agreement group.</span></span>
4. <span data-ttu-id="07557-108">A **Leírás** mezőben adjon meg egy leírást a szolgáltatásiszerződés-csoporthoz.</span><span class="sxs-lookup"><span data-stu-id="07557-108">In the **Description** field, enter a description for the service agreement group.</span></span>
5. <span data-ttu-id="07557-109">A módosítások mentéséhez zárja be az oldalt.</span><span class="sxs-lookup"><span data-stu-id="07557-109">Close the page to save your changes.</span></span>

> [!NOTE]
> <span data-ttu-id="07557-110">Ha meg van határozva egy szolgáltatásiszerződés-csoport a **Szolgáltatáskezelés paraméterei** lap **Szolgáltatásiszerződés-csoport** mezőjében, akkor a program ezt a csoportot ajánlja fel az összes létrehozott új szolgáltatási szerződésnél.</span><span class="sxs-lookup"><span data-stu-id="07557-110">If a service agreement group is defined in the **Service agreement group** field of the **Service management parameters** page, the program suggests this group for all new service agreements that you create.</span></span>
