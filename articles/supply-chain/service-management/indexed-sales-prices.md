# <a name="indexed-sales-prices"></a><span data-ttu-id="55f9a-103">Indexelt eladási árak</span><span class="sxs-lookup"><span data-stu-id="55f9a-103">Indexed sales prices</span></span>  

[!include [banner](../includes/banner.md)]


<span data-ttu-id="55f9a-104">Amikor előfizetési díjat hoz létre, az árindexet állítja be egy előfizetés eladási árához.</span><span class="sxs-lookup"><span data-stu-id="55f9a-104">You set up the index for a subscription sales price when you create a subscription fee.</span></span>

<span data-ttu-id="55f9a-105">Az **Előfizetési díj létrehozása** képernyőn az **Árképzés beolvasása innen** mezőt állítsa **Indexelt alapár** értékre, majd szorozza meg az alapárat a **Százalékos árváltozás** mezőben szereplő százalékos értékkel ahhoz, hogy megkapja az előfizetési tranzakció eladási árát.</span><span class="sxs-lookup"><span data-stu-id="55f9a-105">In the **Create subscription fee** form, set the **Get pricing from** field to **Indexed base price**, and then multiply the base price by the percentage in the **Percent price change** field to get the sales price of the subscription transaction.</span></span>

<span data-ttu-id="55f9a-106">Ha például az alapár 1000 euró, az index pedig 110, az eladási ár 1100 euró lesz.</span><span class="sxs-lookup"><span data-stu-id="55f9a-106">For example, if the base price is EUR 1,000, and the index is 110, the sales price is EUR 1,100.</span></span>
