[!include [banner](../includes/banner.md)]


<span data-ttu-id="c0b40-104">A **Visszárurendelés törlése** képernyőn egyszerre törölhet több visszárurendelést.</span><span class="sxs-lookup"><span data-stu-id="c0b40-104">Use the **Delete return order** form to delete multiple return orders.</span></span>

1.  <span data-ttu-id="c0b40-105">Kattintson az **Értékesítés és marketing**\>**Rendszeres**\>**Adattisztítás**\>**Visszárurendelések törlése** lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="c0b40-105">Click **Sales and marketing** \> **Periodic** \> **Clean up** \> **Delete return orders**.</span></span>

2.  <span data-ttu-id="c0b40-106">Kattintson a **Kiválasztás** gombra az **Értékesítés frissítése** képernyő megnyitásához.</span><span class="sxs-lookup"><span data-stu-id="c0b40-106">Click **Select** to open the **Sales update** form.</span></span>

3.  <span data-ttu-id="c0b40-107">Adja meg a feltételeket, majd kattintson az **OK** gombra a **Visszárurendelések törlése** képernyőre való visszatéréshez.</span><span class="sxs-lookup"><span data-stu-id="c0b40-107">Specify the criteria, and then click **OK** to return to the **Delete return order** form.</span></span> <span data-ttu-id="c0b40-108">A megadott feltételek alapján jelennek meg a jogi személyben létrehozott visszárurendelések.</span><span class="sxs-lookup"><span data-stu-id="c0b40-108">Return orders that have been created in the legal entity are displayed, based on the specified criteria.</span></span>

5.  <span data-ttu-id="c0b40-110">Kattintson az **Igen** gombra a visszárurendelések törléséhez.</span><span class="sxs-lookup"><span data-stu-id="c0b40-110">Click **Yes** to delete the return orders.</span></span>
