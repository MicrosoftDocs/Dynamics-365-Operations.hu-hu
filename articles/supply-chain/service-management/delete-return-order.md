[!include [banner](../includes/banner.md)]

1.  <span data-ttu-id="18e4d-105">Kattintson az **Értékesítés és marketing**\>**Rendszeres**\>**Adattisztítás**\>**Visszárurendelések törlése** lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="18e4d-105">Click **Sales and marketing** \> **Periodic** \> **Clean up** \> **Delete return orders**.</span></span>

3.  <span data-ttu-id="18e4d-107">Adja meg a feltételeket, majd kattintson az **OK** gombra a **Visszárurendelések törlése** képernyőre való visszatéréshez.</span><span class="sxs-lookup"><span data-stu-id="18e4d-107">Specify the criteria, and then click **OK** to return to the **Delete return order** form.</span></span> <span data-ttu-id="18e4d-108">A megadott feltételek alapján jelennek meg a jogi személyben létrehozott visszárurendelések.</span><span class="sxs-lookup"><span data-stu-id="18e4d-108">Return orders that have been created in the legal entity are displayed, based on the specified criteria.</span></span>

5.  <span data-ttu-id="18e4d-110">Kattintson az **Igen** gombra a visszárurendelések törléséhez.</span><span class="sxs-lookup"><span data-stu-id="18e4d-110">Click **Yes** to delete the return orders.</span></span>
