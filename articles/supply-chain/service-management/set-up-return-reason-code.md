## <a name="set-up-return-reason-codes"></a><span data-ttu-id="40939-109">Visszaadási okkódok beállítása</span><span class="sxs-lookup"><span data-stu-id="40939-109">Set up return reason codes</span></span> 

<span data-ttu-id="40939-110">Ez az eljárás segít a visszaadási okkódok beállításában, melyekkel jelezheti, hogy a vevő miért küldte vissza a terméket.</span><span class="sxs-lookup"><span data-stu-id="40939-110">This procedure helps you set up return reason codes that you can use to indicate why a product was returned by the customer.</span></span>

1.  <span data-ttu-id="40939-111">Lépjen az **Értékesítés és marketing** \> **Beállítás** \> **Értékesítési rendelések** \> **Visszaadás** \> **Visszaadási okkódok** lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="40939-111">Go to **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Return reason codes**.</span></span>

5.  <span data-ttu-id="40939-115">A **Visszaadási okkódok csoportjai** legördülő lista segítségével adja hozzá ezt a kódot egy visszajuttatási okkódcsoporthoz.</span><span class="sxs-lookup"><span data-stu-id="40939-115">Use the **Return reason code group** drop-down list to add the code to a return reason code group.</span></span>
