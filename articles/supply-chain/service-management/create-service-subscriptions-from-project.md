[!include [banner](../includes/banner.md)]


<span data-ttu-id="28ad4-104">Szolgáltatási előfizetést egy adott projektből a **Projektvezetés és könyvelés** területen tud létrehozni.</span><span class="sxs-lookup"><span data-stu-id="28ad4-104">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="28ad4-105">Kattintson a következőkre: **Projektvezetés és könyvelés** \> **Közös** \> **Projektek** \> **Minden projekt**.</span><span class="sxs-lookup"><span data-stu-id="28ad4-105">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="28ad4-106">Nyissa meg azt a projektet, amelyhez előfizetést kíván létrehozni.</span><span class="sxs-lookup"><span data-stu-id="28ad4-106">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="28ad4-107">A **Projektek** képernyőn, a **Műveleti ablakban**, a **Kezelés** lapon kattintson a **Szolgáltatás** \> **Szolgáltatási előfizetés** lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="28ad4-107">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

3.  <span data-ttu-id="28ad4-108">Az **Előfizetés** képernyőn, a **Szolgáltatási előfizetés** mezőbe írjon be egy egyedi azonosító számot.</span><span class="sxs-lookup"><span data-stu-id="28ad4-108">On the **Subscription** form, in the **Service subscription** field, enter a unique identification number.</span></span> <span data-ttu-id="28ad4-109">A projektszámot a rendszer automatikusan beírja a kiválasztott projekt előfizetésébe.</span><span class="sxs-lookup"><span data-stu-id="28ad4-109">The project number is entered automatically into the subscription for the selected project.</span></span>

4.  <span data-ttu-id="28ad4-110">Adja meg a többi adatot is az űrlapon.</span><span class="sxs-lookup"><span data-stu-id="28ad4-110">Enter the remaining information in the form.</span></span>

<span data-ttu-id="28ad4-111">Most létrehozhat előfizetési tranzakciókat az előfizetéshez.</span><span class="sxs-lookup"><span data-stu-id="28ad4-111">You can now create subscription transactions for the subscription.</span></span>


> [!NOTE]
> <P><span data-ttu-id="28ad4-112">Ha beállította az előfizetési csoport számsorozatát, akkor a program automatikusan generálja az előfizetés azonosítóját az előfizetés-csoport megadásakor.</span><span class="sxs-lookup"><span data-stu-id="28ad4-112">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
