[!include [banner](../includes/banner.md)]


<span data-ttu-id="f1465-104">Szolgáltatási előfizetést egy adott projektből a **Projektvezetés és könyvelés** területen tud létrehozni.</span><span class="sxs-lookup"><span data-stu-id="f1465-104">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="f1465-105">Kattintson a következőkre: **Projektvezetés és könyvelés** \> **Közös** \> **Projektek** \> **Minden projekt**.</span><span class="sxs-lookup"><span data-stu-id="f1465-105">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="f1465-106">Nyissa meg azt a projektet, amelyhez előfizetést kíván létrehozni.</span><span class="sxs-lookup"><span data-stu-id="f1465-106">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="f1465-107">A **Projektek** képernyőn, a **Műveleti ablakban**, a **Kezelés** lapon kattintson a **Szolgáltatás** \> **Szolgáltatási előfizetés** lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="f1465-107">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

4.  <span data-ttu-id="f1465-110">Adja meg a többi adatot is az űrlapon.</span><span class="sxs-lookup"><span data-stu-id="f1465-110">Enter the remaining information in the form.</span></span>

<span data-ttu-id="f1465-111">Most létrehozhat előfizetési tranzakciókat az előfizetéshez.</span><span class="sxs-lookup"><span data-stu-id="f1465-111">You can now create subscription transactions for the subscription.</span></span>


> [!NOTE]
> <P><span data-ttu-id="f1465-112">Ha beállította az előfizetési csoport számsorozatát, akkor a program automatikusan generálja az előfizetés azonosítóját az előfizetés-csoport megadásakor.</span><span class="sxs-lookup"><span data-stu-id="f1465-112">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
