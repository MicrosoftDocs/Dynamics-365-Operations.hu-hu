# <a name="set-up-service-intervals"></a><span data-ttu-id="67147-103">Szolgáltatás intervallumainak beállítása</span><span class="sxs-lookup"><span data-stu-id="67147-103">Set up service intervals</span></span>  

[!include [banner](../includes/banner.md)]

<span data-ttu-id="67147-104">A szerviz intervalluma azt a gyakoriságot jelzi, amellyel a szervizrendeléssorok létrejönnek a szervizrendelések automatikus létrehozása esetén.</span><span class="sxs-lookup"><span data-stu-id="67147-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

1. <span data-ttu-id="67147-105">Kattintson a **Szolgáltatáskezelés** \> **Beállítás** \> **Szolgáltatási szerződések** \> **Szolgáltatás intervallumai** lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="67147-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service intervals**.</span></span>
2. <span data-ttu-id="67147-106">Hozzon létre egy új szolgáltatás-intervallumot.</span><span class="sxs-lookup"><span data-stu-id="67147-106">Create a new service interval.</span></span>
3. <span data-ttu-id="67147-107">Adja meg a szolgáltatás intervallumának azonosítóját és leírását.</span><span class="sxs-lookup"><span data-stu-id="67147-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="67147-108">Válassza ki a tartományt a **Tartomány** mezőben.</span><span class="sxs-lookup"><span data-stu-id="67147-108">In the **Range** field, select the range.</span></span>
5. <span data-ttu-id="67147-109">A **Gyakoriság** mezőben adja meg a gyakoriságot.</span><span class="sxs-lookup"><span data-stu-id="67147-109">In the **Frequency** field, type the frequency.</span></span> <span data-ttu-id="67147-110">A gyakoriság az a tényező, amellyel meg kell szorozni a tartományt a szolgáltatási szerződés intervallumának megállapítása érdekében.</span><span class="sxs-lookup"><span data-stu-id="67147-110">The frequency is the factor by which you must multiply the range to obtain the interval for a service agreement.</span></span>
6. <span data-ttu-id="67147-111">Az **Alt+S** billentyűkombinációval mentse a szolgáltatás intervallumát.</span><span class="sxs-lookup"><span data-stu-id="67147-111">Press **Alt+S** to save the service interval.</span></span>
