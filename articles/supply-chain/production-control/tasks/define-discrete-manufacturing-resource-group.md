# <a name="define-discrete-manufacturing-resource-group"></a>Elkülönített gyártási erőforráscsoport meghatározása

[!include [task guide banner](../../includes/task-guide-banner.md)]

Egy erőforráscsoport üzemi erőforrások csoportja, amelyek általában megfelelnek a munkacellák fizikai elrendezésének, amelyet a termelésben a padlón lévő sárga vonalak jelölnek. Ez az eljárás bemutatja, hogyan adjon meg egy erőforráscsoportot a különálló termelésben való használathoz. Ezt a folyamatot az USMF bemutatócégen vagy saját adatai használatával is elvégezheti.

1. Ugrás az erőforráscsoportokra.
2. Kattintson az Új lehetőségre.
3. Írjon be egy értéket az Erőforráscsoport mezőbe.
4. A Leírás mezőben adjon meg egy értéket.
5. A Hely mezőben adjon meg vagy válasszon ki egy értéket.
6. A Termelési egység mezőben adjon meg vagy válasszon ki egy értéket.

## <a name="define-default-operational-parameters"></a>Alapértelmezett működési paraméterek meghatározása
1. Bontsa ki a Művelet szakaszt.
2. Adjon meg egy számot a Selejtszázalék mezőben.
3. A Beállítási kategória mezőben adjon meg vagy válasszon ki egy értéket.
4. A Lefutásiidő-kategória mezőben adjon meg vagy válasszon ki egy értéket.
5. A Műveletütemezés százalék mezőjében adjon meg egy számot.

## <a name="define-operating-hours"></a>Működési óraszám megadása
1. Bontsa ki a Naptárak szakaszt.
2. Kattintson a Hozzáadás gombra.
3. A Naptárak mezőben adjon meg vagy válasszon ki egy értéket.

## <a name="add-operations-resources"></a>Üzemi erőforrások hozzáadása
1. Bontsa ki az Erőforrások szakaszt.
2. Kattintson a Hozzáadás gombra.
3. Az Erőforrások mezőben adjon meg vagy válasszon ki egy értéket.
4. Kattintson a Hozzáadás gombra.
5. Az Erőforrások mezőben adjon meg vagy válasszon ki egy értéket.
6. Keresse meg és jelölje ki a kívánt rekordot a listán.
7. A listában kattintson a kijelölt sorban lévő hivatkozásra.
