# <a name="create-a-formula-by-copying-from-an-existing-formula"></a>Receptúra létrehozása meglévő receptúra másolásával

[!include [banner](../../includes/banner.md)]

Ez az eljárás bemutatja, hogyan lehet gyorsan létrehozni új receptúrát meglévő receptúra másolásával. Ezen eljárás létrehozásához használhatók a USMF, USP2 vagy USPI bemutatócégek.


## <a name="create-a-new-formula"></a>Új receptúra létrehozása
1. Ugrás a Receptúrákhoz.
2. Kattintson az Új lehetőségre.
3. Írjon be egy értéket a Receptúra mezőbe.
4. Írjon be egy értéket a Név mezőbe.
5. A Hely mezőben adjon meg vagy válasszon ki egy értéket.
6. Az Elemcsoport mezőben adjon meg, vagy válasszon ki egy értéket.
7. Kattintson a Mentés gombra.

## <a name="copy-from-an-existing-formula"></a>Másolás meglévő receptúrából
1. Kattintson a Másolás lehetőségre.
2. Az Elemszám mezőben adjon meg, vagy válasszon ki egy értéket.
3. A Képletverzió mezőben adjon meg vagy válasszon ki egy értéket.
4. Kattintson az OK gombra.

## <a name="revise-the-copied-formula"></a>A másolt képlet ellenőrzéséhez.
1. Keresse meg és jelölje ki a kívánt rekordot a listán.
2. Kattintson a Törlés gombra.
3. Kattintson az Igen gombra.

## <a name="approve-formula"></a>Receptúra jóváhagyása
1. Kattintson a Receptúra jóváhagyása pontra.
2. A Jóváhagyó mezőben adjon meg vagy válasszon ki egy értéket.
3. Kattintson az OK gombra.
4. Zárja be a lapot.
