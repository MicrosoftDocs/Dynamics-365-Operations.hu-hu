# <a name="copy-co-products-from-an-existing-formula-version"></a>Társtermékek másolása meglévő receptúraverzióból

[!include [task guide banner](../../includes/task-guide-banner.md)]

Ez az eljárás bemutatja, hogy hogyan másolhat társtermékeket egy már meglévő receptúraverzióból egy másik receptúraverzióba egy kiadott termékre vonatkozóan. Az az előfeltétele, hogy legalább egy receptúraverziót társítani kell a társtermékekkel. Ez az eljárás az USP2 bemutatócéget használja.


## <a name="find-a-released-product"></a>Keressen egy kiadott terméket.
1. Ugrás a Kiadott termékek elemre.
2. Kattintson a Szűrők megjelenítése pontra.
    * A termelési mezőtípust készül hozzáadni a szűrő párbeszédpanelen.  
3. Kattintson a Szűrőmező hozzáadása gombra a Termelés típusa mező hozzáadásához.
    * A következő lépésben manuálisan kell megadni a Receptúrát a Termelés típusa mezőben, mielőtt kiválasztja az Alkalmazás lehetőséget. Ez a lehetőség beállítja a kiadott termékek listáján szereplő szűrőt.  
4. Adja meg manuálisan a Receptúrát a Termelés típusa mezőben.
5. Kattintson az Alkalmaz gombra.

## <a name="select-a-released-product"></a>Válasszon ki egy kiadott terméket.
1. Keresse meg és jelölje ki a kívánt rekordot a listán.
2. Kattintson a Receptúraverziók elemre.
    * Kattintson a Műszaki műveletpanelen a receptúraverziókra.  

## <a name="copy-co-products"></a>Társtermékek másolása
1. A Művelet panelen kattintson a Receptúraverzió elemre.
2. Kattintson a Társtermékek elemre.
3. Kattintson a Másolás lehetőségre.
4. Az Elemszám mezőben adjon meg, vagy válasszon ki egy értéket.
5. A Képletverzió mezőben adjon meg vagy válasszon ki egy értéket.
6. Kattintson az OK gombra.
7. Zárja be a lapot.
