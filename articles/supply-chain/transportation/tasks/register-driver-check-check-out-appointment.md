# <a name="register-driver-check-in-and-check-out-for-an-appointment"></a>Illesztőprogram bejelentkezésének és kijelentkezésének regisztrálása egy találkozóhoz

[!include [banner](../../includes/banner.md)]

Ez az eljárás bemutatja, hogyan kell regisztrálni egy járművezetői bejelentkezést és egy járművezetői kijelentkezést. Ezt általában egy szállítási koordinátor végzi. Az USMF bemutatócég adataiban használhatja ezt az eljárást. Mielőtt elkezdené, a rakományhoz be kell állítani egy találkozót. Találkozó létrehozásához előfeltételként lefuttathatja a „Találkozó létrehozása egy rakomány számára” eljárást.


## <a name="select-an-appointment"></a>Válasszon ki egy találkozót
1. Ugrás a Szállításkezelés > Tervezés > Területi találkozóütemezés > Gépjárművezető be- és kijelentkezése elemre.
2. Válasszon ki egy találkozót.

## <a name="register-driver-check-in"></a>Járművezető bejelentkezésének regisztrálása
1. Kattintson a Járművezető bejelentkezése elemre.
2. Írjon be egy értéket a Vontatvány száma mezőbe.
3. Írjon be egy értéket a Járművezető neve mezőbe.
4. Írjon begy értéket a Gépjármű-vezetői jogosítvány mezőbe.
5. Kattintson az OK gombra.

## <a name="register-driver-check-out"></a>Járművezető kijelentkezésének regisztrálása
1. Kattintson a Járművezető kijelentkezése elemre.
2. Kattintson az OK gombra.
