# <a name="set-up-an-appointment-for-a-load"></a>Találkozó beállítása berakodáshoz

[!include [task guide banner](../../includes/task-guide-banner.md)]

Ez az eljárás bemutatja, hogyan kell beállítani és megtervezni egy területi találkozót a rakomány számára. Ezt általában egy szállítási koordinátor végzi. Az USMF bemutatócég adataiban használhatja ezt az eljárást.


## <a name="select-the-load"></a>Jelölje ki a rakományt
1. Ugorjon a Szállításkezelés > Tervezés > Rakománytervező munkaterület elemre.
2. Törölje a Szállított és bevételezett elrejtése jelölőnégyzet jelölését.
3. A listában jelölje ki a rakományt, amelynek az állapota Szállított.
4. Kattintson a Szállítás lehetőségre.
5. Kattintson a Találkozóütemezés lehetőségre.

## <a name="create-an-appointment"></a>Találkozó létrehozása
1. Kattintson az Új lehetőségre.
2. A Találkozó szabály mezőben adjon meg vagy válasszon ki egy értéket.
3. Kattintson a Mentés gombra.
4. Kattintson az Állapot frissítése lehetőségre.
5. Kattintson a Megerősítés gombra.
6. Kattintson a Mentés gombra.
7. Zárja be a lapot.
