# <a name="configure-wave-processing"></a><span data-ttu-id="73c29-103">A hullámfeldolgozás konfigurálása</span><span class="sxs-lookup"><span data-stu-id="73c29-103">Configure wave processing</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="73c29-104">Ez az útmutató leírja, hogy hogyan állíthatja be azokat a feltételeket, amelyek meghatározzák, hogy milyen munkát hozzon létre a rendszer a raktárra vonatkozóan, a hullám feldolgozásakor, és hogy a hullámokat manuálisan vagy automatikusan dolgozzák-e fel.</span><span class="sxs-lookup"><span data-stu-id="73c29-104">This guide describes how to set up the criteria that determine what work is generated for a warehouse when a wave is processed, and whether waves are processed manually or automatically.</span></span> <span data-ttu-id="73c29-105">A hullámsablonok és a lekérdezések beállításával határozza meg a feltételeket, amelyek megegyeznek az értékesítési rendelés, a termelési rendelés vagy a kanban rendelések kiadott soraival rendelkező hullámmal.</span><span class="sxs-lookup"><span data-stu-id="73c29-105">You specify the criteria by setting up wave templates and queries that match a wave with released lines in sales orders, production orders, or kanban orders.</span></span> <span data-ttu-id="73c29-106">A rendszer a hullám feldolgozását azokban raktárházakban végzi, amelyek a Raktárház kezelési modul funkcióját használják, és nem pedig Készlet kezelési modul funkcióját.</span><span class="sxs-lookup"><span data-stu-id="73c29-106">Wave processing is used in warehouses that use the functionality in the Warehouse management module, and not those that use the functionality in the Inventory management module.</span></span> <span data-ttu-id="73c29-107">Ezt a folyamatot lefuttathatja az USMF bemutatócégen.</span><span class="sxs-lookup"><span data-stu-id="73c29-107">You can run this procedure in demo data company USMF.</span></span>

1. <span data-ttu-id="73c29-108">Ugorjon a Raktárkezelés > Beállítás > Hullám > Hullámsablonok pontra.</span><span class="sxs-lookup"><span data-stu-id="73c29-108">Go to Warehouse management > Setup > Waves > Wave templates.</span></span>
2. <span data-ttu-id="73c29-109">Kattintson az Új lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="73c29-109">Click New.</span></span>
3. <span data-ttu-id="73c29-110">Írjon be egy értéket a Hullámsablon mezőbe.</span><span class="sxs-lookup"><span data-stu-id="73c29-110">In the Wave template name field, type a value.</span></span>
    * <span data-ttu-id="73c29-111">A hullámsablon beállításakor meghatározza azt a sorrendet, amelyben a sablonok megegyeznek az értékesítési rendeléseken, a termelési rendeléseken vagy kanbanokon kiadott sorral.</span><span class="sxs-lookup"><span data-stu-id="73c29-111">When you set up a wave template, you specify the sequence in which the templates will be matched to released lines on sales orders, production orders, or kanbans.</span></span> <span data-ttu-id="73c29-112">Ha egy sor ki van adva a raktárba vagy a termeléshez, azt az első hullámsablont használja, amely megfelel a feltételnek.</span><span class="sxs-lookup"><span data-stu-id="73c29-112">When a line is released to the warehouse or to production, it uses the first wave template that it meets the criteria for.</span></span> <span data-ttu-id="73c29-113">Javasoljuk, hogy a legspecifikusabb feltételekkel kialakított sablont helyezze a lista elejére.</span><span class="sxs-lookup"><span data-stu-id="73c29-113">We recommend that you put templates with the most specific criteria at the top of the list.</span></span> <span data-ttu-id="73c29-114">Minél szélesebbek a feltételek, annál valószínűbb, hogy egy sor teljesíti a feltételeket, és ez ahhoz vezethet, hogy a rendszer nem a megfelelő hullámhoz rendeli a sort.</span><span class="sxs-lookup"><span data-stu-id="73c29-114">The broader the criteria, the more likely it is for a line to meet the criteria, and this could lead to lines being assigned to the wrong wave.</span></span>  
4. <span data-ttu-id="73c29-115">Írjon be egy értéket a Hullámsablon leírása mezőbe.</span><span class="sxs-lookup"><span data-stu-id="73c29-115">In the Wave template description field, type a value.</span></span>
5. <span data-ttu-id="73c29-116">A Hely mezőben adjon meg vagy válasszon ki egy értéket.</span><span class="sxs-lookup"><span data-stu-id="73c29-116">In the Site field, enter or select a value.</span></span>
    * <span data-ttu-id="73c29-117">Az USMF használata esetén választhatja a 2-es helyet.</span><span class="sxs-lookup"><span data-stu-id="73c29-117">If you’re using USMF, you can select site 2.</span></span>  
6. <span data-ttu-id="73c29-118">A Raktár mezőben adjon meg vagy válasszon ki egy értéket.</span><span class="sxs-lookup"><span data-stu-id="73c29-118">In the Warehouse field, enter or select a value.</span></span>
    * <span data-ttu-id="73c29-119">Az USMF használata esetén kiválaszthatja az 24-es raktárt.</span><span class="sxs-lookup"><span data-stu-id="73c29-119">If you’re using USMF, you can select warehouse 24.</span></span>  
7. <span data-ttu-id="73c29-120">Állítsa az Hullámlétrehozás automatizálása mezőt Igen értékre.</span><span class="sxs-lookup"><span data-stu-id="73c29-120">Set the Automate wave creation field to Yes.</span></span>
    * <span data-ttu-id="73c29-121">Válassza ezt a lehetőséget, hogy a rendszer automatikusan hozzon létre hullámot, amikor egy értékesítési rendelés, termelési rendelés vagy kanban kiadásra kerül a raktárba.</span><span class="sxs-lookup"><span data-stu-id="73c29-121">Select this option to automatically create a wave when a sales order, production order, or kanban is released to the warehouse.</span></span>  
8. <span data-ttu-id="73c29-122">Állítsa a Hullám feldolgozása a raktárba történő kiadáskor lehetőséget Igen értékre.</span><span class="sxs-lookup"><span data-stu-id="73c29-122">Set the Process wave at release to warehouse option to Yes.</span></span> 
    * <span data-ttu-id="73c29-123">Ha bejelöli ezt a lehetőséget, a rendszer automatikusan feldolgozza a hullámot és létrehozza a munkát, valahányszor egy sor kiadásra kerül a raktárba.</span><span class="sxs-lookup"><span data-stu-id="73c29-123">Select this option to automatically process the wave and create work when a line is released to the warehouse.</span></span>  
9. <span data-ttu-id="73c29-124">Állítsa az Hullámkiadás automatizálása mezőt Igen értékre.</span><span class="sxs-lookup"><span data-stu-id="73c29-124">Set the Automate wave release option to Yes.</span></span> 
    * <span data-ttu-id="73c29-125">Ezzel a beállítással automatikusan adhatja ki a hullámot.</span><span class="sxs-lookup"><span data-stu-id="73c29-125">Select this option to automatically release the wave.</span></span> <span data-ttu-id="73c29-126">A rendszer létrehozza a kitárolási munkát, ami hozzáférhetővé vélik a mobileszközökön.</span><span class="sxs-lookup"><span data-stu-id="73c29-126">The picking work is created and made available on mobile devices.</span></span>  
10. <span data-ttu-id="73c29-127">Állítsa a Hozzárendelés nyitott hullámokhoz mezőt Igen értékre.</span><span class="sxs-lookup"><span data-stu-id="73c29-127">Set the Assign to open waves option to Yes.</span></span> 
    * <span data-ttu-id="73c29-128">A rendszer a hullámsablon lekérdezésszűrője alapján rendeli hozzá a sorokat a hullámokhoz.</span><span class="sxs-lookup"><span data-stu-id="73c29-128">Lines are assigned to waves based on the query filter for the wave template.</span></span>  
11. <span data-ttu-id="73c29-129">Állítsa a Hullám feldolgozása automatikusan a küszöbértéknél lehetőséget Igen értékre.</span><span class="sxs-lookup"><span data-stu-id="73c29-129">Set the Process wave automatically at threshold option to Yes.</span></span> 
    * <span data-ttu-id="73c29-130">Válassza ezt a lehetőséget, hogy a rendszer automatikusan feldolgozza a hullámot, amikor annak értékei elérik a súlyra, szállítmányra vagy sorokra vonatkozóan a Hullám küszöbértékek mezőcsoportban beállított küszöbértékeket.</span><span class="sxs-lookup"><span data-stu-id="73c29-130">Select this option to automatically process the wave when its values reach the thresholds for weight, shipment, and lines specified in the Wave thresholds field group.</span></span> <span data-ttu-id="73c29-131">Ez a lehetőség csak akkor érhető el, ha a Szállítás beállítás van kiválasztva a Hullámsablon típusa mezőben.</span><span class="sxs-lookup"><span data-stu-id="73c29-131">This option is available only if Shipping is selected in the Wave template type field.</span></span>

    * <span data-ttu-id="73c29-133">Válassza ki ezt a lehetőséget, hogy a rendszer automatiksuan létrehozza és kiadja az igényalapú feltöltési munkát.</span><span class="sxs-lookup"><span data-stu-id="73c29-133">Select this option to create demand-based replenishment work and release it automatically.</span></span> <span data-ttu-id="73c29-134">Hozzá kell adnia az újratöltési hullámmódszert a hullámsablonhoz, majd létrehozni egy Hullámigény típusú újratöltési sablont.</span><span class="sxs-lookup"><span data-stu-id="73c29-134">You must add the replenishment wave method to the wave template, and create a replenishment template of the type Wave demand.</span></span>

    * <span data-ttu-id="73c29-136">Az alapvető hullámfeldolgozási módok lehetővé teszik a tevékenységek sorozatát, amelyen minden hullám áthalad, amikor feldolgozta a rendszer.</span><span class="sxs-lookup"><span data-stu-id="73c29-136">Wave template methods allow you to control the sequence of activities that each wave is going through when it’s processed.</span></span> <span data-ttu-id="73c29-137">Például rendelkezhet hullám feltöltési móddal.</span><span class="sxs-lookup"><span data-stu-id="73c29-137">For example, you might have a method for wave replenishment.</span></span> <span data-ttu-id="73c29-138">Egy módszer hozzáadásakor az automatikusan a megfelelő helyen jelenik meg a lépések sorrendjében.</span><span class="sxs-lookup"><span data-stu-id="73c29-138">When you add a method, it’s automatically listed in the appropriate location in the sequence of steps.</span></span> <span data-ttu-id="73c29-139">Ha a Feltöltési munka kiadásának automatizálása beállítást Igen értékre állította be, akkor itt kell hozzáadnia az újratöltési módszert.</span><span class="sxs-lookup"><span data-stu-id="73c29-139">If you’ve set the Automate replenishment work release option to Yes, you need to add the replenish method here.</span></span>  
    * <span data-ttu-id="73c29-140">A hullámattribútumok szűrőként működnek azon típusú cikkek korlátozásához, amelyek a hullámokat használhatják.</span><span class="sxs-lookup"><span data-stu-id="73c29-140">Wave attributes act as filters, to restrict the kind of items that can use the wave.</span></span> <span data-ttu-id="73c29-141">Például megadhatja a egy cikkcsoportot.</span><span class="sxs-lookup"><span data-stu-id="73c29-141">For example, you could specify an item group.</span></span>  
14. <span data-ttu-id="73c29-142">Kattintson a Mentés gombra.</span><span class="sxs-lookup"><span data-stu-id="73c29-142">Click Save.</span></span>
15. <span data-ttu-id="73c29-143">Zárja be a lapot.</span><span class="sxs-lookup"><span data-stu-id="73c29-143">Close the page.</span></span>

17. <span data-ttu-id="73c29-145">Bontsa ki a Hullámfeldolgozás szakaszt.</span><span class="sxs-lookup"><span data-stu-id="73c29-145">Expand the Wave processing section.</span></span>
