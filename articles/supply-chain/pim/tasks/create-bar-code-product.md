# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="299fe-103">Vonalkód létrehozása termékhez</span><span class="sxs-lookup"><span data-stu-id="299fe-103">Create a bar code for a product</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="299fe-104">Ez az témakör bemutatja, hogyan hozható létre manuálisan vonalkód, példaként az M0001 cikkszám használatával.</span><span class="sxs-lookup"><span data-stu-id="299fe-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="299fe-105">Ez az eljárás az USMF bemutatócéget használja.</span><span class="sxs-lookup"><span data-stu-id="299fe-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="299fe-106">Válassza a **Kiadott termék karbantartása** lehetőséget a kezdőlapon.</span><span class="sxs-lookup"><span data-stu-id="299fe-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="299fe-107">Válassza a **Termékek > Kiadott termékek** lehetőséget a **Hivatkozások** szakaszban.</span><span class="sxs-lookup"><span data-stu-id="299fe-107">Go to **Products > Released products** under the **Links** section.</span></span>
3. <span data-ttu-id="299fe-108">Keresse meg és jelölje ki a kívánt rekordot a listán.</span><span class="sxs-lookup"><span data-stu-id="299fe-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="299fe-109">Ebben a példában válassza ki az **M0001** cikkszámot.</span><span class="sxs-lookup"><span data-stu-id="299fe-109">For this example, select item number **M0001**.</span></span>
4. <span data-ttu-id="299fe-110">A Művelet panelen kattintson a **Készletkezelés** elemre.</span><span class="sxs-lookup"><span data-stu-id="299fe-110">On the Action Pane, select **Manage inventory**.</span></span>
5. <span data-ttu-id="299fe-111">Válassza a **Vonalkódok** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="299fe-111">Select **Bar codes**.</span></span>
6. <span data-ttu-id="299fe-112">Válassza az **Új** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="299fe-112">Select **New**.</span></span>
7. <span data-ttu-id="299fe-113">A kiválasztott sort jelölje meg, amely az alábbi listából jön létre.</span><span class="sxs-lookup"><span data-stu-id="299fe-113">Mark the selected row that is created in the list below.</span></span>
8. <span data-ttu-id="299fe-114">A **Vonalkódbeállítás** mezőben adjon meg vagy válasszon ki egy értéket.</span><span class="sxs-lookup"><span data-stu-id="299fe-114">In the **Barcode setup** field, enter or select a value.</span></span>
9. <span data-ttu-id="299fe-115">A **Vonalkód** mezőben adjon meg vagy válasszon ki egy értéket.</span><span class="sxs-lookup"><span data-stu-id="299fe-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="299fe-116">A **Vonalkód** mezőben adjon meg egy értéket.</span><span class="sxs-lookup"><span data-stu-id="299fe-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="299fe-117">Adjon meg egy számot a **Mennyiség** mezőben.</span><span class="sxs-lookup"><span data-stu-id="299fe-117">In the **Quantity** field, enter a number.</span></span>
12. <span data-ttu-id="299fe-118">Válassza a **Mentés** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="299fe-118">Select **Save**.</span></span>
13. <span data-ttu-id="299fe-119">Zárja be a lapot.</span><span class="sxs-lookup"><span data-stu-id="299fe-119">Close the page.</span></span>
