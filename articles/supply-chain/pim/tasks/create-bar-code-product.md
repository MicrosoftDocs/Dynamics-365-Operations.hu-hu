[!include [banner](../../includes/banner.md)]

<span data-ttu-id="2c57e-104">Ez az témakör bemutatja, hogyan hozható létre manuálisan vonalkód, példaként az M0001 cikkszám használatával.</span><span class="sxs-lookup"><span data-stu-id="2c57e-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="2c57e-105">Ez az eljárás az USMF bemutatócéget használja.</span><span class="sxs-lookup"><span data-stu-id="2c57e-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="2c57e-106">Válassza a **Kiadott termék karbantartása** lehetőséget a kezdőlapon.</span><span class="sxs-lookup"><span data-stu-id="2c57e-106">Select **Released product maintenance** on the homepage.</span></span>

9. <span data-ttu-id="2c57e-115">A **Vonalkód** mezőben adjon meg vagy válasszon ki egy értéket.</span><span class="sxs-lookup"><span data-stu-id="2c57e-115">In the **Bar code** field, enter or select a value.</span></span>
