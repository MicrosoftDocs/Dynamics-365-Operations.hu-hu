# <a name="create-an-operating-unit"></a><span data-ttu-id="6db4e-103">Üzemi egység létrehozása</span><span class="sxs-lookup"><span data-stu-id="6db4e-103">Create an operating unit</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="6db4e-104">Egy üzemi egység olyan szervezet, amelynek segítségével meg lehet osztani a gazdasági források és működési folyamatok ellenőrzését egy üzletben.</span><span class="sxs-lookup"><span data-stu-id="6db4e-104">An operating unit is an organization that is used to divide the control of economic resources and operational processes in a business.</span></span> <span data-ttu-id="6db4e-105">Egy üzemi egységben a személyek feladata a szűkös erőforrások maximális kihasználása, a folyamatok javítása és a teljesítményük alapján történő elszámoltatás.</span><span class="sxs-lookup"><span data-stu-id="6db4e-105">People in an operating unit have a duty to maximize the use of scarce resources, improve processes, and account for their performance.</span></span> <span data-ttu-id="6db4e-106">Az üzemi egységek típusai közé tartoznak a költséghelyek, az üzleti egységek, a szervezeti egységek és az érték-előállítási folyamatok.</span><span class="sxs-lookup"><span data-stu-id="6db4e-106">The types of operating units include cost centers, business units, departments, and value streams.</span></span> <span data-ttu-id="6db4e-107">A következő eljárással üzemi egységet hozhat létre.</span><span class="sxs-lookup"><span data-stu-id="6db4e-107">Use the following procedure to create an operating unit.</span></span> <span data-ttu-id="6db4e-108">Ez az eljárás az USMF bemutatócéget használja.</span><span class="sxs-lookup"><span data-stu-id="6db4e-108">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="6db4e-109">Nyissa meg a következőt: **Navigációs lap > Modulok > Szervezeti adminisztráció > Szervezetek > Üzemi egységek**.</span><span class="sxs-lookup"><span data-stu-id="6db4e-109">Go to **Navigation pane > Modules > Organization administration > Organizations > Operating units.**</span></span>
2. <span data-ttu-id="6db4e-110">Az **Új** gombra kattintva nyissa meg a legördülő párbeszédpanelt.</span><span class="sxs-lookup"><span data-stu-id="6db4e-110">Click **New** to open the drop dialog.</span></span>
3. <span data-ttu-id="6db4e-111">Keresse meg és jelölje ki a kívánt rekordot a listán.</span><span class="sxs-lookup"><span data-stu-id="6db4e-111">In the list, find and select the desired record.</span></span> <span data-ttu-id="6db4e-112">Válassza ki a létrehozni kívánt üzemi egységet.</span><span class="sxs-lookup"><span data-stu-id="6db4e-112">Select the type of operating unit you want to create.</span></span>  
4. <span data-ttu-id="6db4e-113">A listában kattintson a kijelölt sorban lévő hivatkozásra.</span><span class="sxs-lookup"><span data-stu-id="6db4e-113">In the list, click the link in the selected row.</span></span>
5. <span data-ttu-id="6db4e-114">Írjon be egy értéket a **Név** mezőbe.</span><span class="sxs-lookup"><span data-stu-id="6db4e-114">In the **Name** field, type a value.</span></span>
    + <span data-ttu-id="6db4e-115">Ha szükséges, bontsa ki az **Általános** területet.</span><span class="sxs-lookup"><span data-stu-id="6db4e-115">Expand the **General** section, if necessary.</span></span>  
    + <span data-ttu-id="6db4e-116">Adja meg az üzemi egység általános adatait, például az azonosítószámot, a DUNS-számot és a vezetőt.</span><span class="sxs-lookup"><span data-stu-id="6db4e-116">Provide general information about the operating unit, such as an identification number, DUNS number, and manager.</span></span>    
    + <span data-ttu-id="6db4e-117">Ha szükséges, bontsa ki a **Címek** területet.</span><span class="sxs-lookup"><span data-stu-id="6db4e-117">Expand the **Addresses** section, if necessary.</span></span>  
    + <span data-ttu-id="6db4e-118">Adja meg a címadatokat, mint például az utcanevet és a házszámot, az irányítószámot és a várost.</span><span class="sxs-lookup"><span data-stu-id="6db4e-118">Enter address information, such as the street name and number, postal code, and city.</span></span> <span data-ttu-id="6db4e-119">A **Hozzáadás** gombra kattintva új címrekordot adhat meg, illetve a Szerkesztés gombra kattintva meglévő címrekordot módosíthat.</span><span class="sxs-lookup"><span data-stu-id="6db4e-119">Click **Add** to enter a new address record, or click Edit to modify an existing address record.</span></span>   
    + <span data-ttu-id="6db4e-120">Bontsa ki a **Kapcsolattartási adatok** szakaszt, ha szükséges.</span><span class="sxs-lookup"><span data-stu-id="6db4e-120">Expand the **Contact information** section, if necessary.</span></span>  
    + <span data-ttu-id="6db4e-121">Adja meg a kommunikációs módszerekkel kapcsolatos adatokat, például e-mail-címeket, URL-címeket és telefonszámokat.</span><span class="sxs-lookup"><span data-stu-id="6db4e-121">Enter information about methods of communication, such as email addresses, URLs, and telephone numbers.</span></span> <span data-ttu-id="6db4e-122">Új kommunikációs rekord megadásához kattintson az Új gombra.</span><span class="sxs-lookup"><span data-stu-id="6db4e-122">To enter a new communication record, click New.</span></span> <span data-ttu-id="6db4e-123">Egy meglévő kommunikációs rekord módosításához kattintson a **További beállítások > Speciális** gombra.</span><span class="sxs-lookup"><span data-stu-id="6db4e-123">To modify an existing communication record, click **More options > Advanced**.</span></span>   
6. <span data-ttu-id="6db4e-124">Igény szerint módosítsa az **Üzemi egység számát**.</span><span class="sxs-lookup"><span data-stu-id="6db4e-124">Optionally, change the **Operating unit number** as needed.</span></span> <span data-ttu-id="6db4e-125">Ne felejtse el, hogy ez a szám egyedi azonosító a vonatkozó **Fél** rekordban, és nem lehet ugyanaz, mint bármely más üzemi egységé.</span><span class="sxs-lookup"><span data-stu-id="6db4e-125">Note that this number is a unique idenitifier for the correspondng **Party** record and cannot be the same as any other operating unit.</span></span>
7. <span data-ttu-id="6db4e-126">Válassza a **Mentés** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="6db4e-126">Select **Save**.</span></span>
