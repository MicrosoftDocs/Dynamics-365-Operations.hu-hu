# <a name="change-the-banner-or-logo"></a><span data-ttu-id="97106-103">A fejléc vagy embléma módosítása</span><span class="sxs-lookup"><span data-stu-id="97106-103">Change the banner or logo</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="97106-104">Az alábbi eljárás felsorolja a lépéseket, amelyeket a rendszergazdák használhatnak a megjelenített szalagcím vagy emblémakép frissítéséhez, ami a jogi személynél jelenik meg.</span><span class="sxs-lookup"><span data-stu-id="97106-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="97106-105">Ez az eljárás az USMF bemutatócéget használja.</span><span class="sxs-lookup"><span data-stu-id="97106-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="97106-106">Ugorjon a Szervezeti adminisztráció > Szervezetek > Jogi személyek pontra.</span><span class="sxs-lookup"><span data-stu-id="97106-106">Go to Organization administration > Organizations > Legal entities.</span></span>
2. <span data-ttu-id="97106-107">A bal oldali listában válassza ki azt a jogi személyt, amelynek módosítani szeretné a szalagcímét vagy emblémáját.</span><span class="sxs-lookup"><span data-stu-id="97106-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="97106-108">Amennyiben már ki van választva, ugorjon a következő lépésre.</span><span class="sxs-lookup"><span data-stu-id="97106-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="97106-109">Bontsa ki a Képek szakaszt.</span><span class="sxs-lookup"><span data-stu-id="97106-109">Expand the Images section.</span></span> <span data-ttu-id="97106-110">Amennyiben már ki van bontva, ugorjon a következő lépésre.</span><span class="sxs-lookup"><span data-stu-id="97106-110">If it is already expanded, go to the next step.</span></span>
4. <span data-ttu-id="97106-111">Kattintson a Szerkesztés lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="97106-111">Click Edit.</span></span>
5. <span data-ttu-id="97106-112">Az Irányítópulton megjelenő vállalati kép típusa mezőben válassza ki: „Szalagcím”.</span><span class="sxs-lookup"><span data-stu-id="97106-112">In the Dashboard company image type field, select 'Banner'.</span></span>
6. <span data-ttu-id="97106-113">Kattintson a Módosítás gombra.</span><span class="sxs-lookup"><span data-stu-id="97106-113">Click Change.</span></span>
    * <span data-ttu-id="97106-114">Tipp: A szalagcím képének ideális felbontása 1920 x 281 képpont.</span><span class="sxs-lookup"><span data-stu-id="97106-114">Tip: The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="97106-115">Az emblémakép ideális szélessége 350 képpont.</span><span class="sxs-lookup"><span data-stu-id="97106-115">The ideal width for a logo image is 350 pixels.</span></span>  
    * <span data-ttu-id="97106-116">A feltöltés befejezésekor megjelenik az új szalagcím vagy embléma képe az Irányítópult képe szakaszban.</span><span class="sxs-lookup"><span data-stu-id="97106-116">When the upload is complete, you should see your new banner or logo image in the Dashboard Image section.</span></span>  
7. <span data-ttu-id="97106-117">Kattintson a Mentés gombra.</span><span class="sxs-lookup"><span data-stu-id="97106-117">Click Save.</span></span>
8. <span data-ttu-id="97106-118">Nyissa meg a következőt: Általános > Általános > Alapértelmezett irányítópult.</span><span class="sxs-lookup"><span data-stu-id="97106-118">Go to Common > Common > Default dashboard.</span></span>
    * <span data-ttu-id="97106-119">Az új szalagcím vagy embléma megjelenik az irányítópulton.</span><span class="sxs-lookup"><span data-stu-id="97106-119">You should see the new banner or logo image on the dashboard.</span></span>  
    * <span data-ttu-id="97106-120">Szükség lehet arra, hogy vállalatát a vállalatválasztó használatával módosítsa arra, amelyiknek feltöltötte a szalagcímét.</span><span class="sxs-lookup"><span data-stu-id="97106-120">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
