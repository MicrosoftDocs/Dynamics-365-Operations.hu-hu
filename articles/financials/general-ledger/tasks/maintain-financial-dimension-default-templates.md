# <a name="maintain-financial-dimension-default-templates"></a><span data-ttu-id="4227e-103">Pénzügyi dimenzió alapértelmezett sablonjainak karbantartása</span><span class="sxs-lookup"><span data-stu-id="4227e-103">Maintain financial dimension default templates</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="4227e-104">Ez az eljárás bemutatja, hogyan állíthatók be a pénzügyi dimenzió alapértelmezett sablonjai.</span><span class="sxs-lookup"><span data-stu-id="4227e-104">This procedure shows how to set up financial dimension default templates.</span></span> <span data-ttu-id="4227e-105">A sablonok segítségével gyorsan szétoszthatók összegek a forrásbizonylatok között.</span><span class="sxs-lookup"><span data-stu-id="4227e-105">The templates can be used to quickly distribute amounts on source documents.</span></span>

1. <span data-ttu-id="4227e-106">Ugorjon a Főkönyv > Számlatükör > Dimenziók > Pénzügyi dimenzió alapvető sablonjai pontra.</span><span class="sxs-lookup"><span data-stu-id="4227e-106">Go to General ledger > Chart of accounts > Dimensions > Financial dimension default templates.</span></span>
2. <span data-ttu-id="4227e-107">Kattintson az Új lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="4227e-107">Click New.</span></span>
3. <span data-ttu-id="4227e-108">A Sablon azonosítója mezőben adja meg a sablon azonosítóját.</span><span class="sxs-lookup"><span data-stu-id="4227e-108">In the Template ID field, enter a Template ID..</span></span>
4. <span data-ttu-id="4227e-109">Adja meg az első dimenzióértékek százalékos értékét.</span><span class="sxs-lookup"><span data-stu-id="4227e-109">Enter the percentage for the first dimension values.</span></span>
5. <span data-ttu-id="4227e-110">Adja meg a megadott százaléknak megfelelő pénzügyidimenzió-értékeket.</span><span class="sxs-lookup"><span data-stu-id="4227e-110">Enter the financial dimension values that correspond to the percent entered.</span></span>
6. <span data-ttu-id="4227e-111">Kattintson a Hozzáadás gombra.</span><span class="sxs-lookup"><span data-stu-id="4227e-111">Click Add.</span></span>
7. <span data-ttu-id="4227e-112">Adjon meg egy másik százalékos értéket a következő dimenzióértéknek.</span><span class="sxs-lookup"><span data-stu-id="4227e-112">Enter another percent for the next dimension values.</span></span>
    * <span data-ttu-id="4227e-113">Ebben a példában adjon meg egy olyan százalékértéket, amivel a felosztott mennyiség 100% alatt marad.</span><span class="sxs-lookup"><span data-stu-id="4227e-113">For this example, enter a percentage so you still have less than 100% allocated.</span></span>  
8. <span data-ttu-id="4227e-114">Adja meg a megadott százaléknak megfelelő pénzügyidimenzió-értékeket.</span><span class="sxs-lookup"><span data-stu-id="4227e-114">Enter the financial dimension values that correspond to the percent entered.</span></span>
9. <span data-ttu-id="4227e-115">Kattintson a Hozzáadás gombra.</span><span class="sxs-lookup"><span data-stu-id="4227e-115">Click Add.</span></span>
10. <span data-ttu-id="4227e-116">Adjon meg egy másik százalékot, amivel az össz-százalék 100% lesz.</span><span class="sxs-lookup"><span data-stu-id="4227e-116">Enter another percent, which will make the total percent equal to 100%.</span></span>
11. <span data-ttu-id="4227e-117">Adja meg a megadott százaléknak megfelelő pénzügyidimenzió-értékeket.</span><span class="sxs-lookup"><span data-stu-id="4227e-117">Enter the financial dimension values that correspond to the percent entered.</span></span>
12. <span data-ttu-id="4227e-118">Kattintson a Mentés gombra.</span><span class="sxs-lookup"><span data-stu-id="4227e-118">Click Save.</span></span>
