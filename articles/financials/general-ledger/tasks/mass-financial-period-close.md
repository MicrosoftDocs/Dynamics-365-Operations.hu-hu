<span data-ttu-id="c81df-104">Ez a cikk bemutatja, hogyan várakoztatható, illetve hogyan zárható le véglegesen egy időszak egyszerre, több jogi személyre vonatkoztatva.</span><span class="sxs-lookup"><span data-stu-id="c81df-104">This topic shows how to place a period on hold or permanently close a period or more than one legal entity at a time.</span></span> <span data-ttu-id="c81df-105">Ezenkívül a feladat megmutatja, hogyan lehet bizonyos modulokra korlátozni a felhasználói csoportok könyvelését.</span><span class="sxs-lookup"><span data-stu-id="c81df-105">In addition, the task will show how to restrict user group posting to specific modules.</span></span>

3. <span data-ttu-id="c81df-110">Válassza ki azt az időszakot, amelynek az állapotát módosítani szeretné.</span><span class="sxs-lookup"><span data-stu-id="c81df-110">Select the period for which you want to modify the status.</span></span>

7. <span data-ttu-id="c81df-118">A **Hozzáférési szintnél** válasszon az **Összes** és az **Egyik sem** lehetőség, illetve egy adott felhasználói csoport között.</span><span class="sxs-lookup"><span data-stu-id="c81df-118">In the **Access** level, select **All**, **None**, or a specific user group.</span></span> <span data-ttu-id="c81df-119">Kattintson a **Kiválasztás** lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="c81df-119">Click **Select**.</span></span> <span data-ttu-id="c81df-120">Az Összes lehetőség jelezi, hogy az összes modulhoz szerkesztési joggal rendelkező felhasználó feladhat, ha az időszak nyitott.</span><span class="sxs-lookup"><span data-stu-id="c81df-120">All indicates all users with edit access to the module can post if the period is open.</span></span> <span data-ttu-id="c81df-121">Az Egyik sem lehetőség azt jelezi, hogy a felhasználók nem adhatnak fel a modulba, ha az időszak nyitott.</span><span class="sxs-lookup"><span data-stu-id="c81df-121">None indicates that users cannot post to the module if the period is open.</span></span> <span data-ttu-id="c81df-122">Egy adott felhasználócsoport lehetőség azt jelzi, hogy csak a csoportba tartozó felhasználók adhatnak fel a modulba, ha az időszak nyitott.</span><span class="sxs-lookup"><span data-stu-id="c81df-122">A specific user group indicates only users in the group are able to post to the module if the period is open.</span></span>

9. <span data-ttu-id="c81df-124">Válasszon másik időszakot állapotfrissítésre.</span><span class="sxs-lookup"><span data-stu-id="c81df-124">Select another period to update the status.</span></span>
10. <span data-ttu-id="c81df-125">Válassza ki a jogi személyeket, amelyek időszakállapotát frissíteni szeretné.</span><span class="sxs-lookup"><span data-stu-id="c81df-125">Select the legal entites for which you want to update the period status.</span></span>
11. <span data-ttu-id="c81df-126">Válassza az **Időszak állapotának frissítése** lehetőséget, és az állapotot állítsa **Várakoztatott**, **Nyitott** vagy **Véglegesen lezárt** értékre.</span><span class="sxs-lookup"><span data-stu-id="c81df-126">Select **Update period status** and set the status of **On hold**, **Open**, or **Permanently closed**.</span></span> <span data-ttu-id="c81df-127">A **Nyitott** lehetőség jelzi, hogy az időszakhoz lehetséges a feladás (ha a felhasználó rendelkezik hozzáféréssel).</span><span class="sxs-lookup"><span data-stu-id="c81df-127">**Open** indicates the period can be posted to, provided the user has access.</span></span> <span data-ttu-id="c81df-128">A **Várakoztatott** érték azt jelenti, hogy az időszak nem adható fel, de újra megnyitható.</span><span class="sxs-lookup"><span data-stu-id="c81df-128">**On hold** means the period cannot be posted to, but the period can be reopened.</span></span> <span data-ttu-id="c81df-129">A **Véglegesen lezárt** azt jelenti, hogy az időszak le van zárva, és többé nem lehet megnyitni.</span><span class="sxs-lookup"><span data-stu-id="c81df-129">**Permanently closed** means the period is closed and can never be opened.</span></span> <span data-ttu-id="c81df-130">Kiigazítások nem adhatók fel.</span><span class="sxs-lookup"><span data-stu-id="c81df-130">Adjustments cannot be posted.</span></span> <span data-ttu-id="c81df-131">Nem ajánlott **véglegesen lezártra** állítani egy időszakot, amíg az összes kiigazítás és vizsgálat le nem zárult.</span><span class="sxs-lookup"><span data-stu-id="c81df-131">We do not recommend setting a period to **Permanently closed** until all adjustments and audits are complete.</span></span>
