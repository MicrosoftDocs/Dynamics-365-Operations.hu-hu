-   <span data-ttu-id="085ea-110">**Automatikus kiegyenlítés** – Állítsa be az **Igen** értéket, ha a tranzakciót automatikusan más nyitott tranzakciókkal szemben kell állítani feladáskor.</span><span class="sxs-lookup"><span data-stu-id="085ea-110">**Automatic settlement** – Set this option to **Yes** if a transaction should be settled automatically against other open transactions when it is posted.</span></span> <span data-ttu-id="085ea-111">Ha a **Nem** értéket állítja be, a felhasználók manuálisan egyenlíthetik ki a tranzakciókat, amikor kifizetést adnak meg, vagy később a **Tranzakciók kiegyenlítése** lap segítségével.</span><span class="sxs-lookup"><span data-stu-id="085ea-111">If this option is set to **No**, users can manually settle transactions when they enter payments, or later, by using the **Settle transactions** page.</span></span>
