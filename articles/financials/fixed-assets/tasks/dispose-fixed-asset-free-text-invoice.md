[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="ef07d-104">Ez az eljárás bemutatja, hogyan értékesíthető egy tárgyi eszköz szabadszöveges számla használatával.</span><span class="sxs-lookup"><span data-stu-id="ef07d-104">This procedure shows how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="ef07d-105">Ugorjon a Kinnlévőségek > Számlák > Kizárólag szabadszöveges számlák pontra.</span><span class="sxs-lookup"><span data-stu-id="ef07d-105">Go to Accounts receivable > Invoices > All free text invoices.</span></span>
2. <span data-ttu-id="ef07d-106">Kattintson az Új lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="ef07d-106">Click New.</span></span>
3. <span data-ttu-id="ef07d-107">A Vevőszámla mezőben adjon meg vagy válasszon ki egy értéket.</span><span class="sxs-lookup"><span data-stu-id="ef07d-107">In the Customer account field, enter or select a value.</span></span>
4. <span data-ttu-id="ef07d-108">Ellenőrizze, és szükség szerint módosítsa a számla alapértelmezett dátumát.</span><span class="sxs-lookup"><span data-stu-id="ef07d-108">Validate the default Invoice date and edit if applicable.</span></span>
5. <span data-ttu-id="ef07d-109">Ellenőrizze a hátralévő alapértelmezett fejlécmezőket, például a pénznemét, és ha szükséges, módosítsa.</span><span class="sxs-lookup"><span data-stu-id="ef07d-109">Validate remaining default header fields, such as Currency and edit if applicable.</span></span>

7. <span data-ttu-id="ef07d-111">Adja meg vagy válassza ki a a számlasor fő számláját.</span><span class="sxs-lookup"><span data-stu-id="ef07d-111">Enter or select the Main account for the invoice line.</span></span>
8. <span data-ttu-id="ef07d-112">Ellenőrizze az alapértelmezett áfacsoportot és a cikk áfacsoportját.</span><span class="sxs-lookup"><span data-stu-id="ef07d-112">Validate the default Sales tax group and Item sales tax group.</span></span>
9. <span data-ttu-id="ef07d-113">Írja be az egységárat vagy a tárgyi eszköz értékesítésének összegét.</span><span class="sxs-lookup"><span data-stu-id="ef07d-113">Enter the Unit price or hte Amount of the sale of the fixed asset.</span></span>
10. <span data-ttu-id="ef07d-114">Kattintson az Sor részletei lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="ef07d-114">Click Line details.</span></span>  
11. <span data-ttu-id="ef07d-115">Válassza ki az értékesítendő tárgyi eszköz számát.</span><span class="sxs-lookup"><span data-stu-id="ef07d-115">Select the Fixed asset number to be sold.</span></span>
12. <span data-ttu-id="ef07d-116">Kattintson a Feladás lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="ef07d-116">Click Post.</span></span>
