# <a name="supported-web-browsers-for-lifecycle-services"></a>A Lifecycle Services támogatott webböngészői 

[!include[banner](../includes/banner.md)]

A felhasználók a Microsoft Dynamics következő népszerű böngészők legújabb verzióinak használatával férhetnek hozzá a Lifecyle Services (LCS) szolgáltatáshoz: 

- Microsoft Edge(ajánlott: [Ium alapú perem](https://support.microsoft.com/microsoft-edge/download-the-new-microsoft-edge-based-on-chromium-0f4a3dd7-55df-60f5-739f-00010dba52cf))
- Króm
- Már(1)

A böngésző legfrissebb változatáért látogasson el a szoftver gyártójának webhelyére. Az optimális teljesítmény és az optimális működés érdekében javasoljuk, hogy a modern böngésző legújabb verzióját használja, különösen Microsoft Edge.
