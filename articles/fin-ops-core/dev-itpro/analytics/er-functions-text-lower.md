# <a name="LOWER">LOWER ER-függvény</a>

[!include [banner](../includes/banner.md)]

A `LOWER` függvény *Karakterlánc* értékként adja vissza a megadott karakterláncot, miután a program kisbetűssé alakította.

## <a name="syntax"></a>Szintaxis

```vb
LOWER (text)
```

## <a name="arguments"></a>Argumentumok

`text`: *Karakterlánc*

Az eredeti szöveget jelölő *Karakterlánc* érték.

## <a name="return-values"></a>Visszaadott értékek

*Karakterlánc*

Az eredményül kapott szövegérték.

## <a name="example"></a>Példa

A `LOWER ("Sample")` a **"sample"** értéket adja vissza.
