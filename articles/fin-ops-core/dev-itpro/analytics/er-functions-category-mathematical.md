# <a name="list-of-er-functions-in-the-mathematical-category"></a>A matematikai kategóriába tartozó ER-függvények listája

[!include [banner](../includes/banner.md)]

Az Elektronikus jelentéskészítés (ER) matematikai függvényei sok általános matematikai számításhoz használhatók. Ez a témakör összefoglalást nyújt a funkciókról.

## <a name="list-of-supported-functions"></a>Támogatott függvények listája

| Funkció | Leírás |
|----------|-------------|
| [(ABS)](er-functions-mathematical-abs.md)             | Ez a függvény a megadott szám abszolútértékét (modulus) adja eredményül *Valós* értékként. Más szóval, a számot előjel nélkül adja vissza. |
| [Teljesítmény](er-functions-mathematical-power.md)         | Ez a függvény egy *Valós* értéket ad eredményül, amely a megadott pozitív számnak a megadott hatványra történő emelésének eredményét jelenti. |
| [Round](er-functions-mathematical-round.md)         | Ez a funkció a megadott számot adja vissza *Valós* értékként, adott számú tizedesjegyre kerekítve. |
| [RoundDown](er-functions-mathematical-rounddown.md) | Ez a funkció a megadott számot adja vissza *Valós* értékként, lefelé kerekítve adott számú tizedesjegyre. |
| [RoundUp](er-functions-mathematical-roundup.md)     | Ez a funkció a megadott számot adja vissza *Valós* értékként, felfelé kerekítve adott számú tizedesjegyre. |
