# <a name="FileDestinationType">Fájl célja</a>

[!include [banner](../includes/banner.md)]

Beállíthat egy fájlcélt a kimenő dokumentumok létrehozásához konfigurált elektronikus jelentési (ER) formátumú fájlok MAPPA vagy FÁJL összetevőihez. A célhely beállítása alapján a webböngészőből letölthető egy generált dokumentum.

A **Célhely beállításai** oldalon a konfiguráció lefutása után a megnyitás/mentés párbeszédpanel akkor jelenik meg, ha az **Engedélyezve** opciót **Igen** állapotba állította.
