# <a name="upper-er-function"></a>UPPER ER-függvény

[!include [banner](../includes/banner.md)]

Az `UPPER` függvény *Karakterlánc* értékként adja vissza a megadott karakterláncot, miután a program nagybetűssé alakította.

## <a name="syntax"></a>Szintaxis

```vb
UPPER (text )
```

## <a name="arguments"></a>Argumentumok

`text`: *Karakterlánc*

A *Karakterlánc* típus adatforrásának érvényes elérési útja.

## <a name="return-values"></a>Visszaadott értékek

*Karakterlánc*

Az eredményül kapott szövegérték.

## <a name="example"></a>Példa

Az `UPPER ("Sample")` a **"SAMPLE"** értéket adja vissza.
