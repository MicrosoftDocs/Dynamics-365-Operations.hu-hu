# <a name="power-bi-destination"></a>Power BI cél

[!include [banner](../includes/banner.md)]

Beállíthat egy archiválási Microsoft Power BI célhelyet a kimenő dokumentumok létrehozásához konfigurált elektronikus jelentési (ER) formátumú fájlok MAPPA vagy FÁJL összetevőihez. A célhely beállítása alapján a generált dokumentumokat a rendszer egy korábban konfigurált SharePoint-mappában tárolja.

Állítsa **Igen** beállításra **az** ER-konfiguráció használatával a Dynamics 365 Pénzügy példányról a szolgáltatásokba történő adatátvitelt Microsoft Power BI. Az átvitt fájlok tárolása egy Microsoft SharePoint Server példányon történik, amelyet ennek a célnak megfelelően konfigurálni kell. További információ: [Elektronikus jelentéskészítés (ER) konfigurálása az adatok Power BI szolgáltatásba való lehívásához](general-electronic-reporting-report-configuration-get-data-powerbi.md).
