# <a name="change-the-banner-or-logo"></a><span data-ttu-id="2b906-103">A fejléc vagy embléma módosítása</span><span class="sxs-lookup"><span data-stu-id="2b906-103">Change the banner or logo</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="2b906-104">Az alábbi eljárás felsorolja a lépéseket, amelyeket a rendszergazdák használhatnak a megjelenített szalagcím vagy emblémakép frissítéséhez, ami a jogi személynél jelenik meg.</span><span class="sxs-lookup"><span data-stu-id="2b906-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="2b906-105">Ez az eljárás az USMF bemutatócéget használja.</span><span class="sxs-lookup"><span data-stu-id="2b906-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="2b906-106">Nyissa meg a következőt: **Navigációs lap > Modulok > Szervezeti adminisztráció > Szervezetek > Jogi személyek**.</span><span class="sxs-lookup"><span data-stu-id="2b906-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="2b906-107">A bal oldali listában válassza ki azt a jogi személyt, amelynek módosítani szeretné a szalagcímét vagy emblémáját.</span><span class="sxs-lookup"><span data-stu-id="2b906-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="2b906-108">Amennyiben már ki van választva, ugorjon a következő lépésre.</span><span class="sxs-lookup"><span data-stu-id="2b906-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="2b906-109">Az **Irányítópult kép** lap kibontása.</span><span class="sxs-lookup"><span data-stu-id="2b906-109">Expand **Dashboard image** tab.</span></span>
4. <span data-ttu-id="2b906-110">**Változtatás** kijelölése.</span><span class="sxs-lookup"><span data-stu-id="2b906-110">Select **Change**.</span></span>
    
    <span data-ttu-id="2b906-111">A szalagcím képének ideális felbontása 1920 x 281 képpont.</span><span class="sxs-lookup"><span data-stu-id="2b906-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="2b906-112">Az emblémakép ideális szélessége 350 képpont.</span><span class="sxs-lookup"><span data-stu-id="2b906-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="2b906-113">Válassza a **Mentés** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="2b906-113">Select **Save**.</span></span>
6. <span data-ttu-id="2b906-114">Ugrás ide: **Navigációs lap > Modulok > Közös > Közös > Alapértelmezett irányítópult**.</span><span class="sxs-lookup"><span data-stu-id="2b906-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="2b906-115">Az új szalagcím vagy embléma megjelenik az irányítópulton.</span><span class="sxs-lookup"><span data-stu-id="2b906-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="2b906-116">Szükség lehet arra, hogy vállalatát a vállalatválasztó használatával módosítsa arra, amelyiknek feltöltötte a szalagcímét.</span><span class="sxs-lookup"><span data-stu-id="2b906-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
