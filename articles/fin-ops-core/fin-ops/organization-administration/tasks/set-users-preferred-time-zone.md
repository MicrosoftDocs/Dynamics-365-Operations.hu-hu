# <a name="set-a-users-preferred-time-zone"></a>Felhasználó preferált időzónájának beállítása

[!include [banner](../../includes/banner.md)]

A következő témakör bemutatja, hogyan állíthatja be egy rendszergazda szerepkörű felhasználó az időzónát egy felhasználó számára.

1. Ugorjon a **Navigációs ablaktábla > Modulok > Rendszerfelügyelet > Felhasználók > Felhasználók** lehetőségre.
2. Keresse meg és jelölje ki a kívánt rekordot a listán.
3. Válassza a **Felhasználói beállítások** elemet.
4. Válassza ki a **Beállítások** lapot.
5. Az **Időzóna** mezőben válasszon ki egy opciót a legördülő listából.
6. Válassza a **Mentés** lehetőséget.
