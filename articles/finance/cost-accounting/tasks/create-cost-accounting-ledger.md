[!include [banner](../../includes/banner.md)]

<span data-ttu-id="b86fc-104">A költségkönyvelési főkönyv megfelel a teljes adatszolgáltatási egységnek.</span><span class="sxs-lookup"><span data-stu-id="b86fc-104">A cost accounting ledger represents the overall reporting unit.</span></span> <span data-ttu-id="b86fc-105">A következők határozzák meg: költségösszetevő-dimenzió, statisztikai dimenzió, pénzügyi naptár és pénznem.</span><span class="sxs-lookup"><span data-stu-id="b86fc-105">It is defined by a cost element dimension, statistical dimension, fiscal calendar, and currency.</span></span> <span data-ttu-id="b86fc-106">Független a jogi személyek fogalmától.</span><span class="sxs-lookup"><span data-stu-id="b86fc-106">It is agnostic to the concept of legal entities.</span></span> <span data-ttu-id="b86fc-107">A jogi személy és annak adatai számos költségkönyvelési főkönyvvel társíthatók.</span><span class="sxs-lookup"><span data-stu-id="b86fc-107">A legal entity and its data can be associated with many cost accounting ledgers.</span></span> <span data-ttu-id="b86fc-108">Ez a felvétel az USP2 bemutató vállalati adatait használja.</span><span class="sxs-lookup"><span data-stu-id="b86fc-108">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="b86fc-109">Lépjen a Költségkönyvelés > Főkönyv beállításai > Költségkönyvelési főkönyvek lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="b86fc-109">Go to Cost accounting > Ledger setup > Cost accounting ledgers.</span></span>
2. <span data-ttu-id="b86fc-110">Kattintson az Új lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="b86fc-110">Click New.</span></span>
3. <span data-ttu-id="b86fc-111">Írjon be egy értéket a Név mezőbe.</span><span class="sxs-lookup"><span data-stu-id="b86fc-111">In the Name field, type a value.</span></span>
4. <span data-ttu-id="b86fc-112">A Költségösszetevő-dimenzió mezőben adjon meg vagy válasszon ki egy értéket.</span><span class="sxs-lookup"><span data-stu-id="b86fc-112">In the Cost element dimension field, enter or select a value.</span></span>
5. <span data-ttu-id="b86fc-113">A Pénzügyi naptár mezőben adjon meg vagy válasszon ki egy értéket.</span><span class="sxs-lookup"><span data-stu-id="b86fc-113">In the Fiscal calendar field, enter or select a value.</span></span>
6. <span data-ttu-id="b86fc-114">A Könyvelési pénznem mezőben írjon be vagy adjon meg egy értéket.</span><span class="sxs-lookup"><span data-stu-id="b86fc-114">In the Accounting currency field, enter or select a value.</span></span>
7. <span data-ttu-id="b86fc-115">Az Árfolyamtípus mezőben adjon meg vagy válasszon ki egy értéket.</span><span class="sxs-lookup"><span data-stu-id="b86fc-115">In the Exchange rate type field, enter or select a value.</span></span>
8. <span data-ttu-id="b86fc-116">A Statisztikai dimenzió mezőben adjon meg vagy válasszon ki egy értéket.</span><span class="sxs-lookup"><span data-stu-id="b86fc-116">In the Statistical dimension field, enter or select a value.</span></span>
9. <span data-ttu-id="b86fc-117">Kattintson a Mentés gombra.</span><span class="sxs-lookup"><span data-stu-id="b86fc-117">Click Save.</span></span>
