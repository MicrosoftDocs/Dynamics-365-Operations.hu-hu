# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="bd35e-103">Tárgyi eszköz kivezetése szabadszöveges számlával</span><span class="sxs-lookup"><span data-stu-id="bd35e-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="bd35e-104">Ez a témakör bemutatja, hogyan értékesíthető egy tárgyi eszköz szabadszöveges számla használatával.</span><span class="sxs-lookup"><span data-stu-id="bd35e-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="bd35e-105">A navigációs panelen ugorjon ide: **Modulok > Kinnlévőségek > Számlák > Kizárólag szabadszöveges számlák**.</span><span class="sxs-lookup"><span data-stu-id="bd35e-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="bd35e-106">Válassza az **Új** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="bd35e-106">Select **New**.</span></span>
3. <span data-ttu-id="bd35e-107">A **Vevőszámla** mezőben adjon meg vagy válasszon ki egy értéket.</span><span class="sxs-lookup"><span data-stu-id="bd35e-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="bd35e-108">Ellenőrizze, és szükség szerint módosítsa a **Számla** alapértelmezett dátumát.</span><span class="sxs-lookup"><span data-stu-id="bd35e-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="bd35e-109">Ellenőrizze a hátralévő alapértelmezett fejlécmezőket, például a **Pénznemet**, és ha szükséges, módosítsa.</span><span class="sxs-lookup"><span data-stu-id="bd35e-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="bd35e-110">A **Számlasorok** szakaszban ellenőrizze a **Leírás** és a **Fő számla** mezőket.</span><span class="sxs-lookup"><span data-stu-id="bd35e-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="bd35e-111">Ellenőrizze az alapértelmezett **Áfacsoportot** és a **Cikk áfacsoportja** mezőket.</span><span class="sxs-lookup"><span data-stu-id="bd35e-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="bd35e-112">Írja be az **Egységárat** vagy a tárgyi eszköz értékesítésének **összegét**.</span><span class="sxs-lookup"><span data-stu-id="bd35e-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="bd35e-113">Jelölje ki a **Sorrészletek** szakaszt.</span><span class="sxs-lookup"><span data-stu-id="bd35e-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="bd35e-114">A **Tárgyi eszköz** mezőben adjon meg vagy válasszon ki egy értéket.</span><span class="sxs-lookup"><span data-stu-id="bd35e-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="bd35e-115">Válassza a **Feladás** parancsot.</span><span class="sxs-lookup"><span data-stu-id="bd35e-115">Select **Post**.</span></span>
