# <a name="view-an-expense-report"></a><span data-ttu-id="de521-103">Költségjelentés megtekintése</span><span class="sxs-lookup"><span data-stu-id="de521-103">View an expense report</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="de521-104">Lekérdezés futtatásakor tranzakció megjelenítésére a főkönyvből vagy a beszerzés és forrásból előfordulhat, hogy a lekérdezés túl sok információ ad vissza.</span><span class="sxs-lookup"><span data-stu-id="de521-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="de521-105">Ezért érdemes a keresést szűkíteni.</span><span class="sxs-lookup"><span data-stu-id="de521-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="de521-106">A szükséges információ eléréséhez lefúrhat a tranzakcióig, hogy megtekintse a költségjelentést, amelyen megjelent.</span><span class="sxs-lookup"><span data-stu-id="de521-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="de521-107">A költségjelentések esetében a Költséggazdálkodás modulban lefúrhat a tranzakciótól az eredeti költségjelentésig.</span><span class="sxs-lookup"><span data-stu-id="de521-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="de521-108">A költségek teljes összege mellett a tételes tranzakciók, az általános forgalmi adó (áfa) visszatérítésre jogosult tranzakciók és a költségekhez csatolt nyugták is megtekinthetők.</span><span class="sxs-lookup"><span data-stu-id="de521-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="de521-109">A lefúráshoz az eredeti költségjelentésig, amelyen a tranzakció megjelent, válassza ki a tranzakciót, majd válassza az **Eredeti dokumentum** elemet.</span><span class="sxs-lookup"><span data-stu-id="de521-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="de521-110">Az egyes tranzakciók megtekintésekor a tranzakcióhoz kapcsolódó további adatok is megtekinthetők.</span><span class="sxs-lookup"><span data-stu-id="de521-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="de521-111">Egy részletezett szállodaköltség esetében például megtekintheti az olyan költségek részletes listáját, mint az internet-hozzáférés, a telefonhívások és a szobaszerviz.</span><span class="sxs-lookup"><span data-stu-id="de521-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>
