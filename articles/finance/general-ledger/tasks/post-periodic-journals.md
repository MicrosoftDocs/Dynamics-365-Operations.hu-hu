# <a name="post-periodic-journals"></a><span data-ttu-id="acdd2-103">Időszaki naplók közzététele</span><span class="sxs-lookup"><span data-stu-id="acdd2-103">Post periodic journals</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="acdd2-104">Az időszaki naplókat olykor ismétlődő naplónak is nevezik, mivel az összeg, a szöveg és más adatok az időszaki napló beolvasásákor minden alkalommal megismétlődnek.</span><span class="sxs-lookup"><span data-stu-id="acdd2-104">Periodic journals are sometimes called recurring journals because the amount, text, and other information are repeated each time that the periodic journal is retrieved.</span></span> <span data-ttu-id="acdd2-105">Időszaki napló létrehozásakor határozza meg az ismétlődéshez tartozó időszakot (pl. nap vagy hónap).</span><span class="sxs-lookup"><span data-stu-id="acdd2-105">When you create the periodic journal, you specify the period interval for the recurrence, such as days or months.</span></span> <span data-ttu-id="acdd2-106">Ez a feladat útmutató havi ismétlődés időszaki naplót hoz létre.</span><span class="sxs-lookup"><span data-stu-id="acdd2-106">This task guide will create a periodic journal with a monthly recurrence.</span></span>

1. <span data-ttu-id="acdd2-107">Nyissa meg a **Navigációs ablakban a Modulok > Főkönyv > Időszakos feladatok > Időszaki naplók** elemet.</span><span class="sxs-lookup"><span data-stu-id="acdd2-107">Go to **Navigation pane > Modules > General ledger > Periodic tasks > Periodic journals**.</span></span>
2. <span data-ttu-id="acdd2-108">Kattintson az **Új** elemre.</span><span class="sxs-lookup"><span data-stu-id="acdd2-108">Click **New**.</span></span>
3. <span data-ttu-id="acdd2-109">A **Név** mezőben adjon meg vagy válasszon ki egy értéket.</span><span class="sxs-lookup"><span data-stu-id="acdd2-109">In the **Name** field, enter or select a value.</span></span>
4. <span data-ttu-id="acdd2-110">A listában kattintson a kijelölt sorban lévő hivatkozásra.</span><span class="sxs-lookup"><span data-stu-id="acdd2-110">In the list, click the link in the selected row.</span></span>
5. <span data-ttu-id="acdd2-111">Írjon egy értéket a **Leírás** mezőbe.</span><span class="sxs-lookup"><span data-stu-id="acdd2-111">In the **Description** field, type a value.</span></span> <span data-ttu-id="acdd2-112">A leírás a későbbi előhíváskor az Időszaki napló neve lesz, ezért adjon neki megfelelő nevet.</span><span class="sxs-lookup"><span data-stu-id="acdd2-112">The description will be the name of the Periodic journal when retrieved later so make sure to give it a relevant name.</span></span>
6. <span data-ttu-id="acdd2-113">A **Művelet ablaktáblán** kattintson a **Sorok** elemre.</span><span class="sxs-lookup"><span data-stu-id="acdd2-113">On the **Action pane**, click **Lines**.</span></span> <span data-ttu-id="acdd2-114">Egy Időszaki napló általában több naplósort tartalmaz.</span><span class="sxs-lookup"><span data-stu-id="acdd2-114">A periodic journal will typically include several journal lines.</span></span> <span data-ttu-id="acdd2-115">Ez az útmutató azonban most csak egy sort ad hozzá.</span><span class="sxs-lookup"><span data-stu-id="acdd2-115">this task guide will however only add one line.</span></span>
7. <span data-ttu-id="acdd2-116">Adja meg a dátumot a **Dátum** mezőben.</span><span class="sxs-lookup"><span data-stu-id="acdd2-116">In the **Date** field, enter a date.</span></span> <span data-ttu-id="acdd2-117">A **Dátum** mező a napi naplóba történő következő átvitel feladási dátumát tartalmazza.</span><span class="sxs-lookup"><span data-stu-id="acdd2-117">The **Date** field contains the posting date for the next transfer to the daily journal.</span></span> <span data-ttu-id="acdd2-118">A naplókhoz, amelyeket havonta előhív, ajánlott a feladás dátumának használata az adott hónapban, általában az időszak első vagy utolsó dátuma.</span><span class="sxs-lookup"><span data-stu-id="acdd2-118">For journals that will be retrieved each month it is recommended to use the date in the month when it will be posted, typically the first or last date in the period.</span></span> <span data-ttu-id="acdd2-119">A Dátum mezőt üresen is hagyhatja, és megadhatja a napló visszaolvasásának dátumát, az Üres dátum mező használatával.</span><span class="sxs-lookup"><span data-stu-id="acdd2-119">It is possible to leave the Date field blank and to give a date when the journal is retrieved, using the Empty date field.</span></span> <span data-ttu-id="acdd2-120">A mező automatikusan frissül a következő ismétlődő dátumra, amikor a tranzakciókat előhívják.</span><span class="sxs-lookup"><span data-stu-id="acdd2-120">The field is automatically updated to the next recurring date when transactions are retrieved.</span></span> 
8. <span data-ttu-id="acdd2-121">A **Számla** mezőben adja meg a kívánt értékeket.</span><span class="sxs-lookup"><span data-stu-id="acdd2-121">In the **Account** field, specify the desired values.</span></span>
9. <span data-ttu-id="acdd2-122">A **Leírás** mezőben adjon meg vagy válasszon ki egy értéket.</span><span class="sxs-lookup"><span data-stu-id="acdd2-122">In the **Description** field, select a value.</span></span>
10. <span data-ttu-id="acdd2-123">Zárja be a lapot.</span><span class="sxs-lookup"><span data-stu-id="acdd2-123">Close the page.</span></span>
11. <span data-ttu-id="acdd2-124">Adjon meg egy számot a **Tartozik** mezőben.</span><span class="sxs-lookup"><span data-stu-id="acdd2-124">In the **Debit** field, enter a number.</span></span>
12. <span data-ttu-id="acdd2-125">Az **Ellenszámla** mezőben adja meg a kívánt értékeket.</span><span class="sxs-lookup"><span data-stu-id="acdd2-125">In the **Offset account** field, specify the desired values.</span></span>
13. <span data-ttu-id="acdd2-126">Válassza a 'Hónapok' lehetőséget az **Egység** mezőben.</span><span class="sxs-lookup"><span data-stu-id="acdd2-126">In the **Unit** field, select 'Months'.</span></span>
14. <span data-ttu-id="acdd2-127">Az **Egységek száma** mezőben írja be az 1-es számot.</span><span class="sxs-lookup"><span data-stu-id="acdd2-127">In the **Number of units** field, enter '1'.</span></span>
15. <span data-ttu-id="acdd2-128">Adja meg a dátumot az **Utolsó dátum** mezőben.</span><span class="sxs-lookup"><span data-stu-id="acdd2-128">In the **Last date** field, enter a date.</span></span> <span data-ttu-id="acdd2-129">Az utolsó dátum megadása az előző időszakban megakadályozza, hogy az ismétlődő napló tévedésből a helytelen kezdőidőszakban jöjjön létre.</span><span class="sxs-lookup"><span data-stu-id="acdd2-129">Entering last date in the preceding period will prevent that the recurring journal by mistake is created in the wrong starting period.</span></span> <span data-ttu-id="acdd2-130">Az utolsó dátum később minden alkalommal frissül, valahányszor az időszaki naplót a program beolvassa.</span><span class="sxs-lookup"><span data-stu-id="acdd2-130">Last date will later on be updated each time the periodic journal is retrieved.</span></span> 
16. <span data-ttu-id="acdd2-131">Kattintson a **Mentés** gombra.</span><span class="sxs-lookup"><span data-stu-id="acdd2-131">Click **Save**.</span></span>
17. <span data-ttu-id="acdd2-132">Nyissa meg a **Navigációs ablak > Modulok > Főkönyv > Naplóbejegyzések > Általános naplók** elemet.</span><span class="sxs-lookup"><span data-stu-id="acdd2-132">Go to **Navigation pane > Modules > General ledger > Journal entries > General journals**.</span></span>
18. <span data-ttu-id="acdd2-133">Kattintson az **Új** elemre.</span><span class="sxs-lookup"><span data-stu-id="acdd2-133">Click **New**.</span></span>
19. <span data-ttu-id="acdd2-134">A **Név** mezőben adjon meg vagy válasszon ki egy értéket.</span><span class="sxs-lookup"><span data-stu-id="acdd2-134">In the **Name** field, enter or select a value.</span></span>
20. <span data-ttu-id="acdd2-135">Keresse meg és jelölje ki a kívánt rekordot a listán.</span><span class="sxs-lookup"><span data-stu-id="acdd2-135">In the list, find and select the desired record.</span></span>
21. <span data-ttu-id="acdd2-136">A listában kattintson a kijelölt sorban lévő hivatkozásra.</span><span class="sxs-lookup"><span data-stu-id="acdd2-136">In the list, click the link in the selected row.</span></span>
22. <span data-ttu-id="acdd2-137">Írjon egy értéket a **Leírás** mezőbe.</span><span class="sxs-lookup"><span data-stu-id="acdd2-137">In the **Description** field, type a value.</span></span>
23. <span data-ttu-id="acdd2-138">A **Művelet ablaktáblán** kattintson a **Sorok** elemre.</span><span class="sxs-lookup"><span data-stu-id="acdd2-138">On the **Action pane**, click **Lines**.</span></span>
24. <span data-ttu-id="acdd2-139">A **Műveleti ablaktáblán** kattintson az **Időszaknapló** elemre.</span><span class="sxs-lookup"><span data-stu-id="acdd2-139">On the **Action pane**, click **Period journal**.</span></span>
25. <span data-ttu-id="acdd2-140">Kattintson a **Napló beolvasása** menüpontra.</span><span class="sxs-lookup"><span data-stu-id="acdd2-140">Click **Retrieve journal**.</span></span>
26. <span data-ttu-id="acdd2-141">Adjon meg egy dátumot a **Záró dátum** mezőben.</span><span class="sxs-lookup"><span data-stu-id="acdd2-141">In the **To date** field, enter a date.</span></span> <span data-ttu-id="acdd2-142">A **Záró dátum** az időszakos bizonylatsor fordulónapjaként szolgál.</span><span class="sxs-lookup"><span data-stu-id="acdd2-142">The **To date** serves as the cutoff date for the periodic voucher lines.</span></span> <span data-ttu-id="acdd2-143">Az ismétlődés beállításától függően, az Utolsó dátum és a Záró dátum ugyanabban a sorban többször is benne lehet a keletkező naplóban.</span><span class="sxs-lookup"><span data-stu-id="acdd2-143">Based on the recurrence setting, the Last date and To date the same line might be included more than once in the resulting journal.</span></span> <span data-ttu-id="acdd2-144">A Záró dátum automatikusan frissül attól függően, hogy az időszaki sor mikor került utoljára át a naplóba.</span><span class="sxs-lookup"><span data-stu-id="acdd2-144">To date is automatically updated based on  session date of the last time the periodic line was transferred to a journal.</span></span> 
27. <span data-ttu-id="acdd2-145">Az **Időszaki naplószám mezőben** adjon meg, vagy válasszon ki egy értéket.</span><span class="sxs-lookup"><span data-stu-id="acdd2-145">In the **Periodic journal number** field, enter or select a value.</span></span>
28. <span data-ttu-id="acdd2-146">A listában kattintson a kijelölt sorban lévő hivatkozásra.</span><span class="sxs-lookup"><span data-stu-id="acdd2-146">In the list, click the link in the selected row.</span></span>
29. <span data-ttu-id="acdd2-147">Kattintson az **OK** gombra.</span><span class="sxs-lookup"><span data-stu-id="acdd2-147">Click **OK**.</span></span> <span data-ttu-id="acdd2-148">Az időszaknapló készen áll a felülvizsgálatra, jóváhagyás vagy az elküldésre a követelményektől vagy a beállítástól függően.</span><span class="sxs-lookup"><span data-stu-id="acdd2-148">The period journal can now be reviewed, approved or posted depending on requirement and setup.</span></span>
