[!include [banner](../../includes/banner.md)]

<span data-ttu-id="7b0aa-104">Az áfa számítása és postázása a Főkönyvi feladási csoportok lehetőségben megadott fő számlákra történik.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-104">Sales tax is calculated and posted to main accounts that are specified in Ledger posting groups.</span></span> <span data-ttu-id="7b0aa-105">Az egyes áfakódokhoz Főkönyvi postázási csoportok vannak csatolva.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-105">Ledger posting groups are attached to each sales tax code.</span></span> <span data-ttu-id="7b0aa-106">Beállíthat külön főkönyvi feladási csoportot mindegyik áfakódhoz, vagy használhat egyetlen főkönyvi feladási csoportot minden áfakódhoz, továbbá több főkönyvi feladási csoportot is hozzárendelhet az áfakódokhoz.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-106">You can set up individual ledger posting groups for each sales tax code, use one ledger posting group for all sales tax codes or assign multiple ledger posting groups to the sales tax codes.</span></span> <span data-ttu-id="7b0aa-107">Ez a felvétel az DEMF bemutatócéget használja.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-107">This recording uses the DEMF demo company.</span></span> 

1. <span data-ttu-id="7b0aa-108">Ugorjon a **Navigációs ablakpanel > Modulok > Áfa > Főkönyvi feladási csoportok elemre**.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-108">Go to **Navigation pane > Modules > Tax > Setup > Sales tax > Ledger posting groups**.</span></span>
2. <span data-ttu-id="7b0aa-109">Kattintson az **Új** elemre.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-109">Click **New**.</span></span>
3. <span data-ttu-id="7b0aa-110">Adjon meg egy értéket a **Főkönyvi feladási csoport** mezőbe.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-110">In the **Ledger posting group** field, type a value.</span></span>
4. <span data-ttu-id="7b0aa-111">Írjon egy értéket a **Leírás** mezőbe.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-111">In the **Description** field, type a value.</span></span>
5. <span data-ttu-id="7b0aa-112">A **Fizetendő áfa** mezőben válassza ki a fő számlát ahhoz a kimenő áfákhoz, amelyeket az adóhatóságnak kell kifizetni.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-112">In the **Sales tax payable** field, select the main account for outgoing sales taxes that are payable to the tax authority.</span></span> <span data-ttu-id="7b0aa-113">Az Áfát az Adóhatóság szedi be, amikor elad adóköteles termékeket és szolgáltatásokat.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-113">Sales taxes are collected on behalf of the tax authority when you sell taxable goods and services.</span></span>  
6. <span data-ttu-id="7b0aa-114">A **Visszaigényelhető áfa** mezőben válassza ki a fő számlát azokhoz a bejövő áfákhoz, amelyek az adóhatóságtól érkeznek.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-114">In the **Sales tax receivable** field, select the main account for incoming taxes that are received from the tax authority.</span></span> <span data-ttu-id="7b0aa-115">Az Áfát a szállítók szedik be az adóhatóság nevében, amikor adóköteles termékeket és szolgáltatásokat vesz.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-115">Vendors collect taxes on behalf of the tax authority when you buy taxable goods and services.</span></span> <span data-ttu-id="7b0aa-116">Ez a mező nem érhető el, csak ha a **Főkönyvi paraméterek** oldalon be van jelölve a Áfaadózási szabályok alkalmazása opció.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-116">This field is not available if the Apply sales tax taxation rules option is selected in the **General ledger parameters** page.</span></span> <span data-ttu-id="7b0aa-117">Ehelyett a szállítóknak befizetett adók ugyanarra a beszerzési számlára terhelődnek.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-117">Instead, sales taxes that are paid to vendors are debited to the same account as the purchase.</span></span>   
7. <span data-ttu-id="7b0aa-118">Az **Importadó költsége** mezőben válassza a fő számlát azoknak a leírható Importadóknak a feladásához, amelyek nem visszaigényeltek, vagy amelyeket jelentettek az adóhatóságnak a szállítók az EU fordított GST/HST részeként.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-118">In the **Use tax expense** field, select  the main account for posting deductible Use taxes that are not claimed or reported to the tax authority by vendors as part of EU reverse charge GST/HST.</span></span> <span data-ttu-id="7b0aa-119">Az **Importadó** lehetőséget ki kell választani az **Áfakódhoz** abból az **Áfacsoportból**, amelyet a tranzakcióban használnak.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-119">The **Use tax** option must be selected for the **Sales tax code** in the **Sales tax group** that is used in the transaction.</span></span> <span data-ttu-id="7b0aa-120">Ez a mező nem lesz elérhető, csak ha a **Főkönyvi paraméterek** oldalon be van jelölve a **Áfaadózási szabályok alkalmazása** opción.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-120">This field won't be available if the **Apply sales tax taxation rules** option is selected on the **General ledger parameters** page.</span></span>   
8. <span data-ttu-id="7b0aa-121">A **Fizetendő importadó** mezőben válassza ki a fő számlát azokhoz a bemenő Importadókhoz, amelyeket az adóhatóságoknak kell kifizetni.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-121">In the **Use tax payable** field, select the main account for posting incoming Use taxes that are payable to tax authorities.</span></span> <span data-ttu-id="7b0aa-122">Az **Importadó** lehetőséget ki kell választani az **Áfakódban** abból az **Áfacsoportból**, amelyet az **Importadó** feladásához használnak.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-122">The **Use tax** option must be selected in the **Sales tax code** in the **Sales tax group** to post **Use tax**.</span></span> <span data-ttu-id="7b0aa-123">Ha az **Áfaadózási szabályok alkalmazása** lehetőséget választja a **Főkönyv paraméterei** oldalon, akkor az eltérés a tranzakció költségszámlájára kerül.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-123">If **Apply sales tax taxation rules** option is selected in **General ledger parameters** page, the offset is posted to the transaction's expense account.</span></span>   
9. <span data-ttu-id="7b0aa-124">A **Kiegyenlítési számla** mezőben válassza ki a fő számlát, így a **Fizetendő importadó** és **Visszaigényelhető áfa** mezőkben megadott főkönyvi számlák nettó egyenlegét adja fel a rendszer.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-124">In the **Settlement account** field, select the main account that the net balance of the ledger accounts specified in the **Use tax payable** and **Sales tax receivable** fields will be posted.</span></span> <span data-ttu-id="7b0aa-125">Az egyenleg az áfák kiegyenlítése és a feladat feladása lett lefuttatva.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-125">The balance will be created when the sales taxes settle and the posting job has been run.</span></span>  <span data-ttu-id="7b0aa-126">Ha a kiegyenlítés időszaknál az adóhatóság egy szállítói számlához van rendelve, akkor az egyenleg feladása ehelyett a szállítói számlára történik majd.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-126">If the taxing authority for the settlement period is associated with a vendor account, the balance will be posted to the vendor account instead.</span></span>
10. <span data-ttu-id="7b0aa-127">A **Szállítói készpénzfizetési engedmény** mezőben a főkönyvi feladási csoporthoz társított áfakód vonatkozásában válassza ki a fő számlát a készpénzfizetési engedmény feladásához.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-127">In the **Vendor cash discount** field, select the main account to post cash discount for Sales tax codes associated with this Ledger posting group.</span></span> <span data-ttu-id="7b0aa-128">Ez nem kötelező, és ha nincs megadva számla, a **Készpénzfizetési engedmény kódjain** található fő számlát használja a rendszer.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-128">This is optional and if no account is entered,  the main account on **Cash discount codes** will be used.</span></span> <span data-ttu-id="7b0aa-129">Érdemes az egyes **Főkönyvi feladási csoportoknál** különböző számlákat használni, amennyiben fordított áfát használ az Áfacsoportokban a készpénzfizetési engedmény lehetőségnél.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-129">It can be useful to use different accounts per **Ledger posting group** if using the reverse sales tax on cash discount option on Sales tax groups.</span></span>  
11. <span data-ttu-id="7b0aa-130">A **Vevői eset engedmény** mezőben válassza ki a fő számlát a készpénzfizetési engedmény feladásához a **Főkönyvi feladási csoporthoz** társított **Áfakódok** esetében.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-130">In the **Customer case discount** field, select the main account to post cash discount for **Sales tax codes** associated with this **Ledger posting group**.</span></span> <span data-ttu-id="7b0aa-131">Ez válaszható, és ha nincs megadva számla, a **Készpénzfizetési engedmény kódjain** található fő számlát használja a rendszer.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-131">This is optional and if no account is entered, the main account on the **Cash discount codes** will be used.</span></span> <span data-ttu-id="7b0aa-132">Érdemes az egyes **Főkönyvi feladási csoportoknál** különböző számlákat használni, amennyiben fordított áfát használ az  **Áfacsoportokban** a készpénzfizetési engedmény lehetőségnél.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-132">It can be useful to use different accounts per **Ledger posting group** if using the reverse sales tax on cash discount option on **Sales tax groups**.</span></span>  
12. <span data-ttu-id="7b0aa-133">Kattintson a **Mentés** gombra.</span><span class="sxs-lookup"><span data-stu-id="7b0aa-133">Click **Save**.</span></span>
