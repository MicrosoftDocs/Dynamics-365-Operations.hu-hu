<span data-ttu-id="eb110-104">Ez az eljárás naplókat és naplósorokat hoz létre, illetve ellenőriz.</span><span class="sxs-lookup"><span data-stu-id="eb110-104">This procedure creates and validates journals and journal lines.</span></span> <span data-ttu-id="eb110-105">Az USMF bemutatócég használatával kipróbálhatja ezt az eljárást.</span><span class="sxs-lookup"><span data-stu-id="eb110-105">You can try this procedure using the USMF demo company.</span></span>  

1. <span data-ttu-id="eb110-106">Ugorjon a **Főkönyv > Naplóbejegyzések > Általános naplók** pontra.</span><span class="sxs-lookup"><span data-stu-id="eb110-106">Go to **General ledger > Journal entries > General journals**.</span></span>
2. <span data-ttu-id="eb110-107">Kattintson az **Új** elemre.</span><span class="sxs-lookup"><span data-stu-id="eb110-107">Click **New**.</span></span>
3. <span data-ttu-id="eb110-108">A **Név** mezőben kattintson a legördítő nyílra a keresőlista megnyitásához.</span><span class="sxs-lookup"><span data-stu-id="eb110-108">In the **Name** field, click the drop-down button to open the lookup.</span></span>
4. <span data-ttu-id="eb110-109">A kívánt rekord megkeresése és kijelölése a listán</span><span class="sxs-lookup"><span data-stu-id="eb110-109">In the list, find and select the desired record.</span></span>

6. <span data-ttu-id="eb110-111">A **Számla** mezőben adja meg a megfelelő számlát a számla típusának megfelelően.</span><span class="sxs-lookup"><span data-stu-id="eb110-111">In the **Account** field enter an appropriate account based on the Account type.</span></span>

10. <span data-ttu-id="eb110-115">Kattintson az **Érvényesítés** gombra.</span><span class="sxs-lookup"><span data-stu-id="eb110-115">Click **Validate**.</span></span>
11. <span data-ttu-id="eb110-116">Kattintson az **Érvényesítés** gombra.</span><span class="sxs-lookup"><span data-stu-id="eb110-116">Click **Validate**.</span></span>
12. <span data-ttu-id="eb110-117">Kattintson a **Bejegyzés** lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="eb110-117">Click **Post**.</span></span>
13. <span data-ttu-id="eb110-118">Kattintson a **Bizonylat** elemre.</span><span class="sxs-lookup"><span data-stu-id="eb110-118">Click **Voucher**.</span></span>
