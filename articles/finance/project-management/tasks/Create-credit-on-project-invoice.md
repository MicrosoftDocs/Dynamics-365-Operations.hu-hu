[!include [banner](../../includes/banner.md)]

1. <span data-ttu-id="c6642-104">Ugorjon a Projektvezetés és könyvelés > Projektek > Minden projekt pontra.</span><span class="sxs-lookup"><span data-stu-id="c6642-104">Go to Project management and accounting > Projects > All projects.</span></span>

4. <span data-ttu-id="c6642-108">A Művelet panelen kattintson a Kezelés elemre.</span><span class="sxs-lookup"><span data-stu-id="c6642-108">On the Action Pane, click Manage.</span></span> 
5. <span data-ttu-id="c6642-109">Kattintson a Számlanaplók lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="c6642-109">Click Invoice journals.</span></span> 
6. <span data-ttu-id="c6642-110">A jóváíráshoz tartozó tranzakció kiválasztása.</span><span class="sxs-lookup"><span data-stu-id="c6642-110">Select the transaction for credit note.</span></span> <span data-ttu-id="c6642-111">A listában jelölje meg a kiválasztott sort.</span><span class="sxs-lookup"><span data-stu-id="c6642-111">In the list, mark the selected row.</span></span>

8. <span data-ttu-id="c6642-113">Kattintson a Kijelölés jóváíráshoz lehetőségre.</span><span class="sxs-lookup"><span data-stu-id="c6642-113">Click Select for credit note.</span></span> 
9. <span data-ttu-id="c6642-114">A jelölőnégyzetet bejelölve válassza ki a jóváíráshoz tartozó tranzakciót.</span><span class="sxs-lookup"><span data-stu-id="c6642-114">Select the transaction for credit note by marking the check box.</span></span>
10. <span data-ttu-id="c6642-115">Kattintson az OK gombra.</span><span class="sxs-lookup"><span data-stu-id="c6642-115">Click OK.</span></span> 

<span data-ttu-id="c6642-116">A lépés után a jóváírás elérhető lesz a számlajavaslathoz.</span><span class="sxs-lookup"><span data-stu-id="c6642-116">After this step, the Credit Note will be available for Invoice Proposal.</span></span>
