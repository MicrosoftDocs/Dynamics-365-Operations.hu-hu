# <a name="transfer-a-quotation-to-a-project"></a><span data-ttu-id="4d8ad-103">Árajánlat átvitele egy projektbe</span><span class="sxs-lookup"><span data-stu-id="4d8ad-103">Transfer a quotation to a project</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="4d8ad-104">Ha egy projektajánlat állapota **Visszaigazolt**, akkor egy vagy több árajánlati sort át lehet vinni egy új vagy egy meglévő projektbe, az **Átvitel projektbe** varázsló segítségével.</span><span class="sxs-lookup"><span data-stu-id="4d8ad-104">When a project quotation has a status of **Confirmed**, you can transfer some or all the quotation lines to a new or existing project by using the **Transfer to project** wizard.</span></span> 

<span data-ttu-id="4d8ad-105">A varázsló használatához kövesse az alábbi lépéseket.</span><span class="sxs-lookup"><span data-stu-id="4d8ad-105">To use the wizard, follow these steps.</span></span>

1. <span data-ttu-id="4d8ad-106">Ugorjon a **Projektvezetés és könyvelés** > **Árajánlatok** > **Projektajánlatok** pontra.</span><span class="sxs-lookup"><span data-stu-id="4d8ad-106">Go to **Project management and accounting** > **Quotations** > **Project quotations**.</span></span>
2. <span data-ttu-id="4d8ad-107">Válassza ki az átvinni kívánt árajánlatot, és a **Nyomon követés** lapon, a **Módosítás** csoportban, válassza az **Átvitel projektbe** parancsot.</span><span class="sxs-lookup"><span data-stu-id="4d8ad-107">Select the quotation you want to transfer and on the **Follow up** tab, in the **Modify** group, select **Transfer to project**.</span></span>
3. <span data-ttu-id="4d8ad-108">Kövesse az **Árajánlat átvitele projekt** varázslót az árajánlat projekthez társításához és átviteléhez.</span><span class="sxs-lookup"><span data-stu-id="4d8ad-108">Follow the **Transfer quotation to project** wizard to associate and transfer the quotation with a project.</span></span>
