[!include [banner](../includes/banner.md)]

<span data-ttu-id="96b3a-104">A projektvezetők beállíthatnak olyan munkalebontási struktúra (WBS) sablonokat, amelyek felhasználhatók új WBS-projektek létrehozásakor.</span><span class="sxs-lookup"><span data-stu-id="96b3a-104">Project managers can set up Work breakdown structure (WBS) templates that they can apply when they create a WBS for new projects.</span></span> <span data-ttu-id="96b3a-105">A projektvezetők létrehozáskor szerepköröket adhatnak a sablonokhoz.</span><span class="sxs-lookup"><span data-stu-id="96b3a-105">Project managers can add roles when they create a template.</span></span> <span data-ttu-id="96b3a-106">Használja a következő eljárást, hogy szerepet rendeljen egy WBS-sablonhoz. </span><span class="sxs-lookup"><span data-stu-id="96b3a-106">Use the following procedure to assign a role to a WBS template.</span></span>

1. <span data-ttu-id="96b3a-107">Válassza a **Projektvezetés és könyvelés** > **Beállítás** > **Projektek** > **Munkalebontási struktúra sablonjai** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="96b3a-107">Select **Project management and accounting** > **Setup** > **Projects** > **Work breakdown structure templates**.</span></span>
