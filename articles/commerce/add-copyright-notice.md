## <a name="add-a-copyright-notice"></a><span data-ttu-id="22a70-109">Szerzői jogi értesítés hozzáadása</span><span class="sxs-lookup"><span data-stu-id="22a70-109">Add a copyright notice</span></span>

<span data-ttu-id="22a70-110">A következő lépésekkel lehet hozzáadni egy szerzői jogi nyilatkozatot az összes olyan oldal aljához, amely egy adott sablont használ.</span><span class="sxs-lookup"><span data-stu-id="22a70-110">To add a copyright notice to the bottom of every page that uses a specific template, follow these steps.</span></span>

1. <span data-ttu-id="22a70-111">Lépjen a **Töredékek** pontra, majd válassza az **Új** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="22a70-111">Go to **Fragments**, and then select **New**.</span></span>
1. <span data-ttu-id="22a70-112">Az **Új töredék** párbeszédpanelen válassza ki a **Lábléc** modult, és nevezze át a töredéket.</span><span class="sxs-lookup"><span data-stu-id="22a70-112">In the **New fragment** dialog box, select the **Footer** module, and name the fragment.</span></span> <span data-ttu-id="22a70-113">Írja be például a **Lábléc-Szerzői jog** értéket.</span><span class="sxs-lookup"><span data-stu-id="22a70-113">For example, enter **Footer-Copyright**.</span></span>
1. <span data-ttu-id="22a70-114">Válassza ki az **OK** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="22a70-114">Select **OK**.</span></span>
1. <span data-ttu-id="22a70-115">A navigációs panelen válassza ki az válassza ki a **Lábléc** melletti három pontot (**…**) majd válassza a **Modul hozzáadása** elemet.</span><span class="sxs-lookup"><span data-stu-id="22a70-115">In the navigation pane, select the ellipsis button (**...**) next to **Footer**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="22a70-116">A párbeszédpanelen válassza ki a **Lábléc-kategória** elemet, majd kattintson az **OK** gombra.</span><span class="sxs-lookup"><span data-stu-id="22a70-116">In the dialog box, select **Footer category**, and then select **OK**.</span></span>
1. <span data-ttu-id="22a70-117">A navigációs panelen válassza ki az válassza ki a **Lábléc-kategória** melletti három pontot, majd válassza a **Modul hozzáadása** elemet.</span><span class="sxs-lookup"><span data-stu-id="22a70-117">In the navigation pane, select the ellipsis button next to **Footer category**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="22a70-118">A párbeszédpanelen válassza ki a **Szövegblokk** elemet, majd kattintson az **OK** gombra.</span><span class="sxs-lookup"><span data-stu-id="22a70-118">In the dialog box, select **Text block**, and then select **OK**.</span></span>
1. <span data-ttu-id="22a70-119">A navigációs ablakban válassza ki a **Szövegblokk** elemet.</span><span class="sxs-lookup"><span data-stu-id="22a70-119">In the navigation pane, select **Text block**.</span></span>
1. <span data-ttu-id="22a70-120">A jobb oldali Tulajdonságok ablaktáblában, a **Bekezdés** mezőbe vegye fel a szerzői joggal kapcsolatos üzenetet.</span><span class="sxs-lookup"><span data-stu-id="22a70-120">In the properties pane on the right, in the **Paragraph** field, add your copyright message.</span></span> <span data-ttu-id="22a70-121">Írja be például a következőt: **(C) Fabrikam 2019**.</span><span class="sxs-lookup"><span data-stu-id="22a70-121">For example, enter **(C) Fabrikam 2019**.</span></span>
1. <span data-ttu-id="22a70-122">Válassza a **Mentés** parancsot, majd a **Szerkesztés befejezése** elemet végül a **Közzététel** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="22a70-122">Select **Save**, select **Finish editing**, and then select **Publish**.</span></span>
1. <span data-ttu-id="22a70-123">Nyissa meg a **Sablonokat** elemet, válasszon ki egy sablont, majd válassza a **Szerkesztés** parancsot.</span><span class="sxs-lookup"><span data-stu-id="22a70-123">Go to **Templates**, select the template, and then select **Edit**.</span></span>
1. <span data-ttu-id="22a70-124">Az **Oldalstruktúra** alatt bontsa ki a **Szövegtörzs** elemet, majd az **Alapértelmezett oldal** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="22a70-124">Under **Page Outline**, expand **Body**, and then expand **Default Page**.</span></span>
1. <span data-ttu-id="22a70-125">Válassza ki a **Lábléchely** hely melletti három pont gombot, majd válassza a **Töredék hozzáadása** elemet.</span><span class="sxs-lookup"><span data-stu-id="22a70-125">Select the ellipsis button next to **Footer Slot**, and then select **Add Fragment**.</span></span>
1. <span data-ttu-id="22a70-126">Válassza ki a korábban létrehozott töredéket, majd válassza a **Kiválasztás** parancsot.</span><span class="sxs-lookup"><span data-stu-id="22a70-126">Select the fragment that you created earlier, and then select **Select**.</span></span>
1. <span data-ttu-id="22a70-127">Válassza a **Szerkesztés befejezése** parancsot a sablon ellenőrzéséhez, majd a **Közzététel** elemet a közzétételhez.</span><span class="sxs-lookup"><span data-stu-id="22a70-127">Select **Finish editing** to check in the template, and then select **Publish** to publish it.</span></span>

<span data-ttu-id="22a70-128">A szerzői jogi nyilatkozatot tartalmazó lábléc automatikusan megjelenik a kijelölt sablont használó oldalak alján.</span><span class="sxs-lookup"><span data-stu-id="22a70-128">The footer that contains the copyright notice automatically appears at the bottom of all pages that use the selected template.</span></span>
