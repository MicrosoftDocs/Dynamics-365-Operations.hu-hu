# <a name="configure-the-functionality-profile-for-a-sales-representative"></a>Funkcióprofil konfigurálása értékesítési képviselőnek

[!include [banner](../includes/banner.md)]

Ez az eljárás bemutatja, hogyan konfigurálhatja egy áruház üzletkötőkre vonatkozó funkcióprofil-beállításait az USRT bemutatóadat-vállalat használatával.

1. Ugorjon a következő elemre: Kiskereskedelem és kereskedelem > Csatorna beállítása > Pénztárbeállítás > Pénztárprofilok > Funkcióprofilok.
2. Kattintson a Szerkesztés lehetőségre.
3. Bontsa ki a Funkciók szakaszt.
    * A profilbeállítások funkció segítségével a pénztár konfigurálható úgy, hogy automatikusan hozzáadja a pénztáros alapértelmezett értékesítési csoportját, hogy bekérje az értékesítési csoportot, illetve hogy kötelező legyen az értékesítési csoport.  
4. Az Üzletkötő megadásának kérése mezőben válasszon egy beállítást.
5. Válassza az Igen lehetőséget az Üzletkötő megkövetelése mezőben.
