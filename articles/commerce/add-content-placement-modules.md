## <a name="overview"></a><span data-ttu-id="39780-105">Áttekintés</span><span class="sxs-lookup"><span data-stu-id="39780-105">Overview</span></span>

<span data-ttu-id="39780-106">A tartalomelhelyezési modul egy speciális tároló, amelybe a többi modult egy konkrét elrendezésben helyezhető el.</span><span class="sxs-lookup"><span data-stu-id="39780-106">A content placement module is a special container that other modules can be put inside in a specific layout.</span></span> <span data-ttu-id="39780-107">A tartalomelhelyezési modulok a következő típusú modulokat támogatják gyermekmodulként: tartalomelhelyezési elem, a fiók üdvözlőképernyő eleme, fiók megrendelési eleme, fiók kívánságlista eleme és fiók profileleme.</span><span class="sxs-lookup"><span data-stu-id="39780-107">Content placement modules support the following types of modules as child modules: content placement item, account welcome item, account order item, account wish list item, and account profile item.</span></span>

<span data-ttu-id="39780-110">A tartalomelhelyezési elem moduljai képekkel és szöveggel mutatják be a promóciókat és termékjellemzőket.</span><span class="sxs-lookup"><span data-stu-id="39780-110">Content placement item modules use both images and text to showcase promotions, policies, and product features.</span></span> <span data-ttu-id="39780-111">Például egy tartalomelhelyezési modul használható a kezdőlapon az üzletszabályzat és a szállítási információk megjelenítésére.</span><span class="sxs-lookup"><span data-stu-id="39780-111">For example, a content placement item module can be used on a home page to show store policies or shipping information.</span></span> <span data-ttu-id="39780-112">A tartalomelhelyezési elem moduljait a tartalomkezelő rendszer (CMS) adatai vezérlik, és bármilyen lapra elhelyezhetők.</span><span class="sxs-lookup"><span data-stu-id="39780-112">Content placement item modules are driven by data from the content management system (CMS) and can be put on any page.</span></span> <span data-ttu-id="39780-113">Azonban csak a tartalomelhelyezési modulban használhatók.</span><span class="sxs-lookup"><span data-stu-id="39780-113">However, they can be used only inside a content placement module.</span></span>

* <span data-ttu-id="39780-116">A tartalomelhelyezési elemeket tartalmazó tartalomelhelyezési modulok a termékek részletes lapjain a termékjellemzők megjelenítésére használhatók.</span><span class="sxs-lookup"><span data-stu-id="39780-116">Content placement modules that contain content placement item modules can be used on product details pages to showcase product features.</span></span>
