# <a name="whats-new-or-changed-in-dynamics-365-for-talent-core-hr"></a>Új vagy módosult elemek a Dynamics 365 for Talent Core HR szolgáltatásban 

[!include [banner](includes/banner.md)]

Ezen a lap arról tájékoztatja, hogyan találhat információkat a Microsoft Dynamics Talent HR Core legújabb kiadásairól.

Minden héten új kiadás jelenik meg a Talent alkalmazáshoz. Ezen kiadásoka docs.microsoft.com webhelyen található tartalomjegyzékben vannak felsorolva ([docs.microsoft.com/dynamics365/unified-operations](../index.md)). Átnézheti a tartalomjegyzék, vagy a tartalomjegyzék felett található Szűrővel megkeresheti a legújabb Önt érdeklő témákat. A témakörök címe az „Új vagy módosult elemek a Dynamics 365 for Talent…” kifejezéssel kezdődik.

A tartalomjegyzék továbbra is frissítve lesz minden héten, így később is megtalálja a Talent kiadásainak új funkcióit, illetve változásait.
