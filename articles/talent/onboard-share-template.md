## <a name="share-an-onboarding-guide-with-other-contributors"></a><span data-ttu-id="fed58-105">Osszon meg bevezető útmutatókat más közreműködőkkel</span><span class="sxs-lookup"><span data-stu-id="fed58-105">Share an onboarding guide with other contributors</span></span>

1. <span data-ttu-id="fed58-106">Kattintson a bal oldali menü **Guides** parancsára, majd válassza ki azt az útmutatót, amelyet meg szeretne osztani.</span><span class="sxs-lookup"><span data-stu-id="fed58-106">On the left menu, select **Guides**, and then select the guide that you want to share.</span></span>

    ![[<span data-ttu-id="fed58-108">Felvételi útmutató megosztása más közreműködőkkel</span><span class="sxs-lookup"><span data-stu-id="fed58-108">Sharing an onboarding guide with other contributors</span></span>](./media/onboard-share-guide.png)](./media/onboard-share-guide.png)

3. <span data-ttu-id="fed58-109">Ha a **Megosztás vele:** területen a címzetteknek képesnek kell lennie az útmutató felülvizsgálatára, megadhatja az **Engedélyezi ezen személyek számára a szerkesztést?** pontnál a **Be** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="fed58-109">Under **Share with**, if recipients should be able to revise the guide, set the **Can these people make edits?** option to **On**.</span></span> <span data-ttu-id="fed58-110">Ha a címzetteket e-mailben kell értesíteni arról, hogy az útmutatót már megosztotta velük, állítsa az **E-mail-értesítés küldése a címzetteknek** beállítást **Be** értékre.</span><span class="sxs-lookup"><span data-stu-id="fed58-110">If recipients should be notified by email that you've shared the guide with them, set the **Send email notification to recipients** option to **On**.</span></span>

    ![[<span data-ttu-id="fed58-111">Hozzáférés és e-mail-beállítások megadása felvételi útmutatónál</span><span class="sxs-lookup"><span data-stu-id="fed58-111">Setting access and email preferences for an onboarding guide</span></span>](./media/onboard-share-guide-with-contributors.png)](./media/onboard-share-guide-with-contributors.png)

4. <span data-ttu-id="fed58-112">Adjon meg egy nevet vagy e-mail-címet, majd a legördülő listából válassza ki a kívánt címzettet.</span><span class="sxs-lookup"><span data-stu-id="fed58-112">Enter a name or email address, and select the desired recipient in the drop-down list.</span></span>
