## <a name="share-an-onboarding-guide-with-other-contributors"></a><span data-ttu-id="cd84d-105">Osszon meg bevezető útmutatókat más közreműködőkkel</span><span class="sxs-lookup"><span data-stu-id="cd84d-105">Share an onboarding guide with other contributors</span></span>

1. <span data-ttu-id="cd84d-106">Kattintson a bal oldali menü **Guides** parancsára, majd válassza ki azt az útmutatót, amelyet meg szeretne osztani.</span><span class="sxs-lookup"><span data-stu-id="cd84d-106">On the left menu, select **Guides**, and then select the guide that you want to share.</span></span>
2. <span data-ttu-id="cd84d-107">Válassza ki a **Továbbiak** gombot (három függőleges pont) a lap jobb felső sarkában, majd válassza a **Megosztás** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="cd84d-107">Select the **More** button (the three vertical dots) in the upper-right corner of the page, and then select **Share**.</span></span>

    ![[<span data-ttu-id="cd84d-108">Felvételi útmutató megosztása más közreműködőkkel</span><span class="sxs-lookup"><span data-stu-id="cd84d-108">Sharing an onboarding guide with other contributors</span></span>](./media/onboard-share-guide.png)](./media/onboard-share-guide.png)

3. <span data-ttu-id="cd84d-109">Ha a **Megosztás vele:** területen a címzetteknek képesnek kell lennie az útmutató felülvizsgálatára, megadhatja az **Engedélyezi ezen személyek számára a szerkesztést?** pontnál a **Be** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="cd84d-109">Under **Share with**, if recipients should be able to revise the guide, set the **Can these people make edits?** option to **On**.</span></span> <span data-ttu-id="cd84d-110">Ha a címzetteket e-mailben kell értesíteni arról, hogy az útmutatót már megosztotta velük, állítsa az **E-mail-értesítés küldése a címzetteknek** beállítást **Be** értékre.</span><span class="sxs-lookup"><span data-stu-id="cd84d-110">If recipients should be notified by email that you've shared the guide with them, set the **Send email notification to recipients** option to **On**.</span></span>

    ![[<span data-ttu-id="cd84d-111">Hozzáférés és e-mail-beállítások megadása felvételi útmutatónál</span><span class="sxs-lookup"><span data-stu-id="cd84d-111">Setting access and email preferences for an onboarding guide</span></span>](./media/onboard-share-guide-with-contributors.png)](./media/onboard-share-guide-with-contributors.png)

4. <span data-ttu-id="cd84d-112">Adjon meg egy nevet vagy e-mail-címet, majd a legördülő listából válassza ki a kívánt címzettet.</span><span class="sxs-lookup"><span data-stu-id="cd84d-112">Enter a name or email address, and select the desired recipient in the drop-down list.</span></span>
5. <span data-ttu-id="cd84d-113">A **Hozzáférés kezelése** lapon áttekintheti, illetve módosíthatja, hogy mit tehetnek a címzettek az útmutatóval.</span><span class="sxs-lookup"><span data-stu-id="cd84d-113">On the **Manage access** tab, you can review or change what recipients can do with the guide.</span></span> <span data-ttu-id="cd84d-114">A címzett hozzáférésének módosításához a címzett neve melletti legördülő listában válassza ki a **Megtekintés engedélyezve**, **Szerkesztés engedélyezve** vagy az **Eltávolítás** pontot.</span><span class="sxs-lookup"><span data-stu-id="cd84d-114">To change a recipient's access, in the drop-down list next to the recipient's name, select **Can View**, **Can edit**, or **Remove**, as appropriate.</span></span>
6. <span data-ttu-id="cd84d-115">Ha befejezte a címzettek hozzáadását vagy módosítását, válassza a **Frissítés** pontot.</span><span class="sxs-lookup"><span data-stu-id="cd84d-115">When you've finished adding or modifying recipients, select **Update**.</span></span>

4. <span data-ttu-id="cd84d-123">Adjon meg egy nevet vagy e-mail-címet, majd a legördülő listából válassza ki a kívánt címzettet.</span><span class="sxs-lookup"><span data-stu-id="cd84d-123">Enter a name or email address, and select the desired recipient in the drop-down list.</span></span>
