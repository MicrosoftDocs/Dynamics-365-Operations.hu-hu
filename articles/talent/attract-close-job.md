# <a name="close-a-job"></a><span data-ttu-id="fc504-103">Állás lezárása</span><span class="sxs-lookup"><span data-stu-id="fc504-103">Close a job</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="fc504-104">A Microsoft Dynamics 365 Talent: Attract megoldásban **Állás érvénytelenítve** vagy **Beosztás betöltve** értékkel zárhat le egy állást.</span><span class="sxs-lookup"><span data-stu-id="fc504-104">In Microsoft Dynamics 365 Talent: Attract, you can close a job as either **Job cancelled** or **Position filled**.</span></span> <span data-ttu-id="fc504-105">Bármikor érvényteleníthet egy állást.</span><span class="sxs-lookup"><span data-stu-id="fc504-105">You can cancel a job at any time.</span></span>

<span data-ttu-id="fc504-106">Ha jelezni szeretné, hogy a beosztást betöltöttek, egy pályázónak el kell fogadnia az állásajánlatot.</span><span class="sxs-lookup"><span data-stu-id="fc504-106">If you want to indicate that the position was filled, a candidate must have accepted an offer for the job.</span></span>

## <a name="cancel-a-job"></a><span data-ttu-id="fc504-107">Állás érvénytelenítése</span><span class="sxs-lookup"><span data-stu-id="fc504-107">Cancel a job</span></span>

<span data-ttu-id="fc504-108">Bármikor érvényteleníthet egy állást az Attract megoldásban.</span><span class="sxs-lookup"><span data-stu-id="fc504-108">You can cancel a job in Attract at any time.</span></span>

1. <span data-ttu-id="fc504-109">Az Attract **Állások** lapján válassza ki azt az állást, amelyet vissza szeretne vonni.</span><span class="sxs-lookup"><span data-stu-id="fc504-109">In the **Jobs** tab in Attract, select the job you want to cancel.</span></span>

2. <span data-ttu-id="fc504-110">Válassza az **Állás lezárása** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="fc504-110">Select **Close job**.</span></span>

   ![Állása lezárása az Attract megoldásban](./media/attract-close-job.png)

3. <span data-ttu-id="fc504-112">Az **Állás lezárása** párbeszédpanelen válassza az **Állás érvénytelenítve** lehetőséget majd adja meg a megjegyzéseket, majd válassza az **Állás lezárása** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="fc504-112">In the **Close this job** dialog box, select **Job cancelled**, add any comments, and then select **Close job**.</span></span>

   ![Állás lezárása Beosztás betöltve értékkel](./media/attract-close-job-as-cancelled.png)

## <a name="close-a-job-as-position-filled"></a><span data-ttu-id="fc504-114">Egy állás lezárása Beosztás betöltve értékkel</span><span class="sxs-lookup"><span data-stu-id="fc504-114">Close a job as Position filled</span></span>

<span data-ttu-id="fc504-115">Az állást akkor zárhatja le **Beosztás betöltve** értékkel, ha legalább egy potenciális jelölt befejezte az ajánlati szakaszt.</span><span class="sxs-lookup"><span data-stu-id="fc504-115">You can close a job as **Position filled** after at least one prospect has completed the Offer stage.</span></span>

1. <span data-ttu-id="fc504-116">Az Attract **Állások** lapján válassza ki azt az állást, amelyet vissza szeretne vonni.</span><span class="sxs-lookup"><span data-stu-id="fc504-116">In the **Jobs** tab in Attract, select the job you want to cancel.</span></span>

2. <span data-ttu-id="fc504-117">Válassza az **Állás lezárása** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="fc504-117">Select **Close job**.</span></span>

   ![Állása lezárása az Attract megoldásban](./media/attract-close-job.png)

3. <span data-ttu-id="fc504-119">Az **Állás lezárása** párbeszédpanelen válassza ki a **Beosztás betöltve** értéket, válassza ki azt a pályázót, aki elfogadta az ajánlatot, vegyen fel megjegyzést, majd válassza a **Az állás lezárása** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="fc504-119">In the **Close this job** dialog box, select **Position filled**, select the candidate who accepted the offer, add any comments, and then select **Close job**.</span></span>

   ![Állás lezárása Beosztás betöltve értékkel](./media/attract-close-job-as-position-filled.png)

   > [!NOTE]
   > <span data-ttu-id="fc504-121">Az állást csak akkor zárhatja le **Beosztás betöltve** értékkel, ha legalább egy potenciális jelölt befejezte az ajánlati szakaszt.</span><span class="sxs-lookup"><span data-stu-id="fc504-121">You can only close a job as **Position filled** when at least one prospect has completed the Offer stage.</span></span> <span data-ttu-id="fc504-122">Ha az álláspályázathoz nincs kiválasztva az **Ajánlat elfogadva** lehetősége, vagy jelentkezésben nem az **Elfogadott** állapot az ajánlatban, a **Beosztás betöltve** lehetőség nem jelenik meg az **Ok** legördülő menüben.</span><span class="sxs-lookup"><span data-stu-id="fc504-122">If the job application doesn't have **Offer accepted** selected, or if the application doesn't display a status of **Accepted** in Offer, the **Position filled** option won't display in the **Reason** dropdown.</span></span>
