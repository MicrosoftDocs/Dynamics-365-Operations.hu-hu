# <a name="screening-frequency"></a>Szűrési gyakoriság

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Ez a témakör bemutatja a Szűrési gyakoriság beállításkészletet a Dynamics 365 Human Resources rendszerben.

Fizikai név: mshr_hcmfrequencyunit

Ez a felsorolás a szűrési gyakorisághoz kapcsolódó értékek beállításkészletét adja meg. 

| Érték | Címke | Leírás |
| --- | --- | --- |
| 200000000 Csak egyszeri | A szűrésre csak egyszer van szükség. |
| 200000001 Naponta | A szűrési gyakoriság számítása napokban történik. |
| 200000002 Hetente | A szűrési gyakoriság számítása hetekben történik. |
| 200000003 Havonta | A szűrési gyakoriság számítása hónapokban történik. |
| 200000004 Évente | A szűrési gyakoriság számítása években történik. |
