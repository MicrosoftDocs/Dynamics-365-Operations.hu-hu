# <a name="payroll---employment-type-option-set"></a>Bérlista – Foglalkoztatástípus beállítása

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Ez a témakör bemutatja a Foglalkoztatástípus lehetőség beállítását a Dynamics 365 Human Resources rendszerben.

## <a name="option-set-employment-type"></a>Értékkészlet: Foglalkoztatástípus

Ez a felsorolás az alkalmazott foglalkoztatástípusához kapcsolódó beállításkészletet adja meg. Ez a lehetőség a mshr_hcmemploymenttype beállításkészletben érhető el.

| Érték | Címke | Leírás |
| --- | --- | --- |
| 200000000 | Alkalmazott | Alkalmazott |
| 200000001 | Alvállalkozó | Alvállalkozó |
