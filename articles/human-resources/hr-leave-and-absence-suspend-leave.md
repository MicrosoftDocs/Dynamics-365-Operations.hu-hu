# <a name="suspend-leave"></a><span data-ttu-id="52c38-103">Szabadság felfüggesztése</span><span class="sxs-lookup"><span data-stu-id="52c38-103">Suspend leave</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="52c38-104">Felfüggesztheti egy alkalmazottak szabadságát, hogy leállítsa a szabadság-elhatárolások feldolgozását a kiválasztott szabadságtípusokhoz.</span><span class="sxs-lookup"><span data-stu-id="52c38-104">You can suspend leave for an employee to stop leave accruals from being processed for selected leave types.</span></span> 

## <a name="suspend-leave-and-absence-for-an-employee"></a><span data-ttu-id="52c38-105">Egy alkalmazott szabadságának és távollétének felfüggesztése</span><span class="sxs-lookup"><span data-stu-id="52c38-105">Suspend leave and absence for an employee</span></span>

3. <span data-ttu-id="52c38-108">Válassza az **Új** lehetőséget.</span><span class="sxs-lookup"><span data-stu-id="52c38-108">Select **New**.</span></span>

4. <span data-ttu-id="52c38-109">A **Szabadság-elhatárolás felfüggesztése** párbeszédpanelen válassza ki a **Szabadság típust** a felfüggesztés **Kezdő dátuma** és **Záró dátuma** értékekkel együtt.</span><span class="sxs-lookup"><span data-stu-id="52c38-109">In the **Suspend leave accrual** dialog box, select the **Leave type** along with the **Start date** and **End date** for the suspension.</span></span>
