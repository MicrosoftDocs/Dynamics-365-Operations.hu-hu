# <a name="period-unit"></a>Időszakegység

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Ez a témakör bemutatja az Időszak intervalluma lehetőség beállítását a Dynamics 365 Human Resources rendszerben.

Fizikai név: mshr_periodunit

Ez a felsorolás a mértékegységek értékekkel kapcsolatos beállításkészletét adja meg meghatározott időszakokra.

| Érték | Címke | Leírás |
| --- | --- | --- |
| 200000000 | Napi | Nap. |
| 200000001 | Hónap | Hónap. |
| 200000002 | év | Év. |
