# <a name="regulatory-job-category"></a>Jogszabállyal kapcsolatos álláskategória

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Ez a témakör bemutatja Jogszabállyal kapcsolatos álláskategória beállításkészletet a Dynamics 365 Human Resources rendszerben.

Fizikai név: mshr_hcmregulatoryjobcategory

Ez a felsorolás a jogszabállyal kapcsolatos álláskategória beállítási értékeire vonatkozó beállításkészletet határozza meg. Ez a meglévő mshr_hcmregulatoryjobcategory beállításkészletben található.

| Érték | Címke | Leírás |
| --- | --- | --- |
| 200000000 | None | Egyik sem. |
| 200000001 | Vezető | Vezető beosztású hivatalnokok és vezetők. |
| 200000002 | Kezelő | Első/középső szintű hivatalnokok és vezetők. |
| 200000003 | Szakmai | Szakemberek. |
| 200000004 | Technikus | Technikusok. |
| 200000005 | SalesWorker | Értékesítők. |
| 200000006 | Adminisztrációs | Adminisztratív támogatást nyújtó dolgozók. |
| 200000007 | CraftWorker | Kétkezi dolgozók. |
| 200000008 | Operatív | Operatív személyek. |
| 200000009 | Munkás | Munkások/segítők. |
| 200000010 | ServiceWorker | Szervizelési dolgozók. |
