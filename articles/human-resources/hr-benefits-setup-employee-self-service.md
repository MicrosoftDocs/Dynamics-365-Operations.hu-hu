# <a name="configure-employee-self-service"></a>Alkalmazotti önkiszolgálás konfigurálása

[!include [banner](includes/preview-feature.md)]

A Microsoft Dynamics 365 Human Resources alkalmazásban lehetősége van csempéket konfigurálni az Alkalmazotti önkiszolgáló rendszer felső szintű navigációjához. A juttatási terv csempék a felhasználókat olyan juttatási konstrukciókhoz irányítják, amelyekre jogosultak regisztrálni.

## <a name="set-up-a-role-center-tile"></a>Szerepkörfőoldal-csempe beállítása

1. A **Juttatások kezelése** munkaterület **Beállítás** részén válassza az **Alkalmazotti önkiszolgáló rendszer paraméterei** elemet.

2. Válassza ki a **Szerepkörfőoldal-mozaik beállítása** lapot, majd válassza az **Új** lehetőséget.

3. Adjon meg értékeket a következő mezőkben:

   | Mező | Leírás |
   | --- | --- |
   | Csempe azonosítója | A mozaik egyedi azonosítója. |
   | CSempecímke felirata | Az önkiszolgáló rendszer csempéjén megjelenő szöveg. |
   | Leírás | A csempe leírása. |
   | Internetcím | Adja meg az alkalmazotti önkiszolgáló rendszer oldalának URL-jét. |
   | Csempeméret | A csempe mérete: kis, közepes vagy nagy. |
   | Cél | Megadja, hogy a lapnak új ablakban vagy az aktuális ablakban kell-e megnyílnia. |
   | Csempeháttérkép | A csempéhez használandó kép URL-címe (nem kötelező). |
   | Kezdete | A csempe elérhetőségének kezdő dátuma és időpontja. |
   | End | A csempe elérhetőségének befejező dátuma és időpontja. |

4. Válassza a **Mentés** lehetőséget.
