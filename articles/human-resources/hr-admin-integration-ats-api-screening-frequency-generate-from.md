# <a name="screening-frequency-generate-from"></a>Szűrési gyakoriság generálása innen:

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Ez a témakör bemutatja a Szűrési gyakoriság generálása innen: beállításkészletet a Dynamics 365 Human Resources rendszerben.

Fizikai név: mshr_hcmfrequencygeneratefrom

Ez a felsorolás lehetőséget nyújt a következő kötelező szűrés kiszámításának kezdő dátumának meghatározására szolgáló értékek beállításkészletére vonatkozóan.

| Érték | Címke | Leírás |
| --- | --- | --- |
| 200000000 Nincs kiválasztva | Nincs kijelölve érték. |
| 200000001 Befejezés dátuma | A számítás az utolsó szűrés befejezési dátumától történik. |
| 200000002 Elvárt dátum | A számítás az utolsó szűrés elvárt dátumától történik. |
