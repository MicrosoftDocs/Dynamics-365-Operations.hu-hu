# <a name="removed-or-deprecated-features-in-dynamics-365-human-resources"></a>A Dynamics 365 Emberi erőforrások eszköz eltávolított vagy elavult funkciói

Ez a témakör olyan funkciókat ír le, amelyek már el vannak távolítva, Microsoft Dynamics illetve amelyek a 365 Emberi erőforrások eszközből való eltávolításra vannak tervezve.

- Az *eltávolított* szolgáltatások már nem érhetők el a termékben.
- Az *elavult* szolgáltatás már nincs aktív fejlesztés alatt, és egy jövőbeli frissítésben eltávolíthatjuk.

Ez a lista segít figyelembe venni az elavult és eltávolított szolgáltatásokat a saját tervezése elősegítésére.

## <a name="features-removed-or-deprecated-in-human-resources"></a>Az Emberi erőforrások eszközben eltávolított vagy elavult funkciók 

### <a name="new-lcs-project-creation-in-human-resources-stand-alone"></a>Új LCS-projekt létrehozása az Emberi erőforrások eszközben (önálló) 

[Emberi erőforrások létesítása](/hr-admin-setup-provision)

| &nbsp; | &nbsp; |
|------------|--------------------|
| **Elavulás/eltávolítás oka** | Lecserélve a projekt létrehozására a pénzügyi és a műveleti infrastruktúra területén. Az infrastruktúra-egyesítésről [a Dynamics 365 Emberi erőforrások infrastruktúra-egyesítési gyakori kérdések című témakörében olvashat bővebben](/hr-infrastructure-merge-faq). |
| **Felváltotta másik szolgáltatás?**   | Igen |
| **Érintett területek**         | Alkalmazás |
| **Telepítési beállítás**              | Minden |
| **Állapot**                         | Elavult: 2022 júliusában már nem tudjuk támogatni az új projektek létrehozását az Emberi erőforrások (önálló) eszközben. |

### <a name="production-environment-provisioning-on-human-resources-stand-alone"></a>Termelési környezet létesítés az emberi erőforrásokon (önálló) 

| &nbsp; | &nbsp; |
|------------|--------------------|
| **Elavulás/eltávolítás oka** | Lecserélve a pénzügyi és a műveleti infrastruktúra környezetvédelmi szolgáltatásával. Az infrastruktúra-egyesítésről [a Dynamics 365 Emberi erőforrások infrastruktúra-egyesítési gyakori kérdések című témakörében olvashat bővebben](/hr-infrastructure-merge-faq.md). |
| **Felváltotta másik szolgáltatás?**   | Igen |
| **Érintett területek**         | Alkalmazás |
| **Telepítési beállítás**              | Minden |
| **Állapot**                         | Elavult: 2022 júliusában már nem tudjuk támogatni a HR gyártási környezetek létesítését az Emberi erőforrások (önálló). |
