# <a name="award-type"></a>Jutalom típusa


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Ez a témakör a "Award type" beállítás beállításának beállítását írja le Dynamics 365 Human Resources.

## <a name="option-set-award-type"></a>Beállításkészlet: Jutalom típusa

Fizikai név: mshr_HrmCompVarAwardEmplType

Ez a felsorolás biztosítja a beállításkészletet a változó kompenzáció jutalomtípusához a Human Resources virtuális entitásaiban.

| Érték | Címke | Leírás |
| --- | --- | --- |
| 200000000 | Általános jutalom | Általános jutalom. |
| 200000001 | Egyösszegű fix fizetés | Egyösszegű fix fizetés. |
