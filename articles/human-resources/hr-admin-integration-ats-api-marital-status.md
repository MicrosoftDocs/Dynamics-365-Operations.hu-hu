# <a name="marital-status"></a>Családi állapot


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Ez a témakör a családi állapot beállításának beállítását írja le Dynamics 365 Human Resources.

Fizikai név: mshr_dirpersonmaritalstatus

Ez a felsorolás a személyi rekordokhoz kapcsolódó családi állapot beállításkészletét adja meg.

| Érték | Címke | Leírás |
| --- | --- | --- |
| 200000000 | None | Nincs kijelölve érték.
| 200000001 | Egyedülálló | Egyedülálló. |
| 200000002 | Házas | Házas. |
| 200000003 | Elvált | Elvált. |
| 200000004 | Özvegy | Özvegység. |
