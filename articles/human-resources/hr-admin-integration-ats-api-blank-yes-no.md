# <a name="blank-yes-no"></a>Üres Igen Nem

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Ez a témakör bemutatja az Üres Igen Nem lehetőség beállítását a Dynamics 365 Human Resources rendszerben.

Fizikai név: mshr_hrmblankyesno

Ez a felsorolás az igen/nem tulajdonságok beállítását adja meg, amelyek üres karaktereket is engedélyeznek.

| Érték | Címke | Leírás |
| --- | --- | --- |
| 200000000 | Üres | Nincs kijelölve érték. |
| 200000001 | Igen | Igen. |
| 200000002 | Nincs | Szám |
