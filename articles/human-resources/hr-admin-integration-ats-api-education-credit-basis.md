# <a name="education-credit-basis"></a>Végzettség jóváírásának alapja

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Ez a témakör bemutatja Végzettség jóváírásának alapja beállításkészletet a Dynamics 365 Human Resources rendszerben.

Fizikai név: mshr_hcmeducationcreditbasis

Ez a felsorolás a pályázók végzettségével kapcsolatos rekordja végzettség-jóváírási alapját adó értékek beállításkészletét adja meg.

| Érték | Címke | Leírás |
| --- | --- | --- |
| 200000000 | Üres | Nincs kijelölve érték. |
| 200000001 | Félév | Félév. |
| 200000002 | Negyedév | Negyedév. |
| 200000003 | Trimeszter | Trimeszter. |
| 200000004 | Időszak | Időszak. |
| 200000005 | Egyéb | Egyéb. |
