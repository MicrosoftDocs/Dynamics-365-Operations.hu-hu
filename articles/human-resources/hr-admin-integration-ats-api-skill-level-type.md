# <a name="skill-level-type"></a>Szakértelmi szint típusa

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Ez a témakör bemutatja a Szakértelmi szint típusa lehetőség beállítását a Dynamics 365 Human Resources rendszerben.

Fizikai név: mshr_hrmskillleveltype

Ez a felsorolás a jelölt szakértelméhez rendelt szintet kategorizáló értékkészletet teszi elérhetővé.

| Érték | Címke | Leírás |
| --- | --- | --- |
| 200000000 | Tényleges | Tényleges szakértelmi szint. |
| 200000001 | Cél | A szakértelmi szint megcélzott értéke. |
