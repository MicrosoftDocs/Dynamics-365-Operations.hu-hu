# <a name="payroll---no-yes-option-set"></a>Bérlista – Nem Igen beállításkészlet

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Ez a témakör bemutatja a Igen Nem lehetőség beállítását a Dynamics 365 Human Resources rendszerben.

## <a name="option-set-no-yes"></a>Beállításkészlet: Nem Igen

Fizikai név: mshr_noyes

Ez a felsorolás biztosítja a beállításkészletet a Human Resources virtuális entitásainak tipikus logikai tulajdonságaihoz.

| Érték | Címke | Leírás |
| --- | --- | --- |
| 200000000 | Nincs | Nem. |
| 200000001 | Igen | Igen. |
