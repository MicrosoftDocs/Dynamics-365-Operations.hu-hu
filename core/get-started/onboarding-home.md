# <a name="onboarding-home-page"></a>Használat megkezdésének kezdőlapja

[!include[banner](../includes/banner.md)]


A Microsoft Dynamics 365 for Finance and Operations használatának megkezdéséhez kiválaszthatja az Önnek megfelelő előfizetési szintet, kereshet egy tapasztalt partnert a régióján vagy iparágán belül, és használatba veheti a Microsoft FastTrack programot, hogy gyorsabban megkezdhesse a szolgáltatás használatát.  

A következő források áttekintést nyújtanak a Lifecycle Servicesről, illetve arról, hogyan működhet együtt a partnerével a Finance and Operations használatának megkezdésében.
